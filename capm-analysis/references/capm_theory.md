# CAPM 理论参考

## 一、CAPM 简介

**CAPM（资本资产定价模型，Capital Asset Pricing Model）** 是由 William Sharpe（1964）、John Lintner（1965）和 Jan Mossin（1966）独立提出的金融资产定价模型。

CAPM 是**现代投资组合理论（MPT）** 的延伸，用于计算**资产的预期收益率**，给定其**系统风险（Beta）**。

---

## 二、核心公式

### 2.1 CAPM 公式

```
E(R_i) = R_f + β_i × (E(R_m) - R_f)
```

**其中**：

| 符号 | 含义 | 说明 |
|------|------|------|
| `E(R_i)` | 资产 i 的**预期收益率** | 投资者对资产 i 的期望回报 |
| `R_f` | **无风险利率** | 通常指国债收益率（如 10年期国债） |
| `β_i` | 资产 i 的**贝塔系数** | 衡量资产 i 相对于市场组合的系统风险敞口 |
| `E(R_m)` | 市场组合的**预期收益率** | 通常指市场指数（如 S&P 500、沪深300）的预期回报 |
| `E(R_m) - R_f` | **市场风险溢价** | 投资者因承担市场平均风险而要求的额外回报 |

---

### 2.2 Beta（β）系数

**定义**：

```
β_i = Cov(R_i, R_m) / Var(R_m)
```

**计算方式**（线性回归）：

```
R_i - R_f = α_i + β_i × (R_m - R_f) + ε_i
```

其中：
- `R_i`：资产 i 的实际收益率
- `R_m`：市场组合的实际收益率
- `α_i`：阿尔法系数（超额收益）
- `ε_i`：残差项（非系统风险）

**Beta 的解释**：

| Beta 值 | 含义 | 示例资产 |
|---------|------|----------|
| `β > 1` | 资产波动性**大于**市场 | 科技股（如特斯拉 β ≈ 1.5） |
| `β = 1` | 资产波动性**等于**市场 | 市场指数基金（如沪深300 ETF） |
| `0 < β < 1` | 资产波动性**小于**市场 | 公用事业股（如电力股 β ≈ 0.5） |
| `β = 0` | 资产收益与市场**无关** | 无风险资产（如国债） |
| `β < 0` | 资产收益与市场**负相关** | 黄金、看跌期权 |

## 三、CAPM 的假设

CAPM 基于以下**严格假设**（现实中可能不成立）：

### 3.1 投资者假设

1. **理性投资者**：所有投资者都是理性的、风险厌恶的
2. **同质预期**：所有投资者对预期收益率、风险和相关性的判断一致
3. **单期投资**：所有投资者持有相同投资期限

### 3.2 市场假设

4. **无交易成本**：买卖资产无需支付佣金、税费
5. **无税收**：资本利得税、股息税等为 0
6. **完全信息**：所有投资者免费获取相同信息

### 3.3 资产假设

7. **资产无限可分**：可以购买任意数量的资产（如 0.001 股）
8. **无卖空限制**：可以自由卖空任何资产

### 3.4 借贷假设

9. **无限制借贷无风险资产**：可以以无风险利率无限制地借入或贷出资金

## 四、CAPM 的局限性

### 4.1 理论局限性

1. **假设过于严格**：现实中交易成本、税收、卖空限制等存在
2. **市场组合不可观测**：CAPM 中的"市场组合"应包括**所有**资产（股票、债券、房地产、人力资本等），但实际中只能用市场指数（如 S&P 500）代替
3. **Beta 不稳定性**：Beta 可能随时间变化（结构性断裂）

### 4.2 实证局限性

1. **Fama-French 三因子模型**：研究发现，除 Beta 外，**市值因子**（小盘股 vs 大盘股）和**账面市值比因子**（价值股 vs 成长股）也能解释收益率差异
2. **低 Beta 异象**：现实中，低 Beta 股票的收益率往往**高于**CAPM 预测（Beta 低估）
3. **非系统风险溢价**：CAPM 假设只有系统风险（Beta）有溢价，但现实中非系统风险也可能有溢价

## 五、CAPM 的应用场景

### 5.1 投资决策

**场景**：判断股票是否被高估或低估

**方法**：
1. 计算股票的**预期收益率（CAPM）**
2. 计算股票的**实际预期收益率**（基于分析师预测或历史数据）
3. 比较：
   - 如果 `实际预期收益率 > CAPM 预期收益率` → 股票**被低估**（买入）
   - 如果 `实际预期收益率 < CAPM 预期收益率` → 股票**被高估**（卖出）

---

### 5.2 投资组合管理

**场景**：构建投资组合，平衡风险与收益

**方法**：
1. 计算组合中每只股票的 Beta
2. 计算组合的**加权平均 Beta**：
   ```
   β_portfolio = Σ(w_i × β_i)
   ```
   其中 `w_i` 是股票 i 的权重
3. 根据组合的 Beta，调整风险敞口：
   - 如果 `β_portfolio > 1` → 组合波动性大于市场（高风险）
   - 如果 `β_portfolio < 1` → 组合波动性小于市场（低风险）

---

### 5.3 资本成本计算

**场景**：计算公司的**加权平均资本成本（WACC）**

**方法**：
1. 使用 CAPM 计算**股权资本成本**：
   ```
   r_e = R_f + β_e × (E(R_m) - R_f)
   ```
   其中 `β_e` 是公司的 Beta
2. 计算**债务资本成本**（`r_d`）
3. 计算 WACC：
   ```
   WACC = (E/V) × r_e + (D/V) × r_d × (1 - T)
   ```
   其中：
   - `E`：股权价值
   - `D`：债务价值
   - `V = E + D`：公司总价值
   - `T`：公司税率

---

### 5.4 业绩归因

**场景**：评估基金经理的业绩

**方法**：
1. 计算基金的 **Alpha（α）**：
   ```
   α = R_actual - [R_f + β × (R_m - R_f)]
   ```
2. 解释：
   - `α > 0`：基金经理创造了**超额收益**（战胜市场）
   - `α = 0`：基金经理获得了**市场平均收益**（与风险匹配）
   - `α < 0`：基金经理获得了**低于市场的收益**（未战胜市场）

## 六、CAPM 的扩展模型

### 6.1 Fama-French 三因子模型

**公式**：

```
E(R_i) - R_f = α_i + β_i × (R_m - R_f) + s_i × SMB + h_i × HML + ε_i
```

**其中**：

| 因子 | 含义 | 解释 |
|------|------|------|
| `R_m - R_f` | 市场风险溢价 | 市场组合收益率 - 无风险利率 |
| `SMB` | 小盘股溢价 | Small Minus Big（小盘股收益率 - 大盘股收益率） |
| `HML` | 价值股溢价 | High Minus Low（高账面市值比收益率 - 低账面市值比收益率） |

---

### 6.2 Fama-French 五因子模型

**公式**：

```
E(R_i) - R_f = α_i + β_i × (R_m - R_f) + s_i × SMB + h_i × HML + r_i × RMW + c_i × CMA + ε_i
```

**新增因子**：

| 因子 | 含义 | 解释 |
|------|------|------|
| `RMW` | 盈利溢价 | Robust Minus Weak（高盈利收益率 - 低盈利收益率） |
| `CMA` | 投资溢价 | Conservative Minus Aggressive（低投资增长率 - 高投资增长率） |

---

### 6.3 Carhart 四因子模型

**公式**：

```
E(R_i) - R_f = α_i + β_i × (R_m - R_f) + s_i × SMB + h_i × HML + m_i × UMD + ε_i
```

**新增因子**：

| 因子 | 含义 | 解释 |
|------|------|------|
| `UMD` | 动量溢价 | Up Minus Down（过去表现好的股票收益率 - 过去表现差的股票收益率） |

## 七、A股市场的 CAPM 应用注意事项

### 7.1 市场基准选择

**推荐基准**：

| 基准 | 代码 | 适用场景 |
|------|------|----------|
| 沪深300 | `000300.SH` | 大盘蓝筹股 |
| 中证500 | `000905.SH` | 中盘成长股 |
| 中证1000 | `000852.SH` | 小盘股 |

**注意**：不同基准会导致不同的 Beta 估计，建议尝试多个基准。

---

### 7.2 无风险利率选择

**推荐**：

| 期限 | 代码 | 适用场景 |
|------|------|----------|
| 10年期 | 国债收益率 | 长期投资 |
| 1年期 | 国债收益率 | 中期投资 |
| 3个月 | 国债收益率 | 短期投资 |

**获取方式**：
- `westock-data` skill（推荐）
- `akshare` 库（`bond_zh_us_rate()`）

---

### 7.3 涨跌停处理

**问题**：A股有涨跌停限制（±10%），可能导致收益率计算偏差

**解决方案**：
1. 使用**复权价格**（前复权 / 后复权）
2. 剔除**涨跌停日**的异常值
3. 使用**周度或月度收益率**（降低微观结构噪声）

---

### 7.4 非交易天数处理

**问题**：A股有春节、国庆等长假，导致收益率序列不连续

**解决方案**：
1. 使用**交易日历**（如 `pandas_market_calendars` 库）
2. 计算收益率时，使用**实际交易日**的数据

## 九、Python 实现要点

### 9.1 使用 `statsmodels.OLS`（推荐）

```python
import statsmodels.api as sm

# 计算超额收益率
stock_excess = stock_returns - risk_free_rate / 252
market_excess = market_returns - risk_free_rate / 252

# 添加常数项（用于估计 Alpha）
X = sm.add_constant(market_excess)
y = stock_excess

# 回归
model = sm.OLS(y, X).fit()

# 提取结果
beta = model.params['market_excess']
alpha = model.params['const'] * 252  # 转换为年化
r_squared = model.rsquared
p_value = model.pvalues['market_excess']
```

---

### 9.2 使用 `numpy.polyfit`（不推荐）

```python
import numpy as np

# 计算超额收益率
stock_excess = stock_returns - risk_free_rate / 252
market_excess = market_returns - risk_free_rate / 252

# 回归（一阶多项式 = 线性回归）
beta, alpha = np.polyfit(market_excess, stock_excess, 1)

# 注意：此方法无法计算 R²、p 值、标准误等诊断指标
```

---

### 9.3 计算预期收益率

```python
# CAPM 公式
expected_return = risk_free_rate + beta * (market_return - risk_free_rate)

# 其中：
# - risk_free_rate：无风险利率（年化，小数形式）
# - beta：贝塔系数
# - market_return：市场组合预期收益率（年化，小数形式）
```

## 十、常见问题

### Q1：Beta 越高越好吗？

**A**：不一定。

- **高 Beta**（> 1）：在市场上涨时收益更高，但在市场下跌时损失更大
- **低 Beta**（< 1）：在市场上涨时收益更低，但在市场下跌时损失更小

**选择取决于**：
- 市场展望（牛市 → 高 Beta，熊市 → 低 Beta）
- 风险偏好（激进 → 高 Beta，保守 → 低 Beta）

---

### Q2：CAPM 适用于A股吗？

**A**：适用，但需要注意：

1. **市场基准选择**：沪深300、中证500 等
2. **无风险利率**：使用中国国债收益率
3. **涨跌停处理**：使用复权价格
4. **模型局限性**：A股市场有效性较弱，CAPM 的解释力度（R²）可能较低

---

### Q3：如何判断 CAPM 模型是否可靠？

**A**：查看以下诊断指标：

1. **R²**：拟合优度，越高越好（通常 > 0.3 可接受）
2. **Beta p 值**：应 < 0.05（Beta 显著不为 0）
3. **残差分析**：残差应近似服从正态分布、无异方差性
4. **稳健性检验**：使用不同时间段、不同市场基准，观察 Beta 是否稳定

---

### Q4：CAPM 和 multi-factor-scoring 有什么关系？

**A**：CAPM 的 Beta 可以作为**风险因子**集成到多因子打分系统。

**示例**：

```
多因子打分 = w1 × Value因子 + w2 × Momentum因子 + w3 × Beta因子 + ...
```

**其中**：
- Beta 因子 = Beta 值（越高 = 系统风险越大 = 预期收益率越高）

## 十一、总结

**CAPM 的核心思想**：

> 投资者只因承担**系统风险（Beta）** 而获得风险溢价。非系统风险可以通过多样化投资消除，因此没有溢价。

**CAPM 的贡献**：

1. 提供了计算**预期收益率**的理论框架
2. 将风险分解为**系统风险**和**非系统风险**
3. 为**资本成本计算**、**投资组合管理**、**业绩归因**等提供了理论基础

**CAPM 的局限**：

1. 假设过于严格（无交易成本、同质预期等）
2. 无法解释所有收益率差异（需要多因子模型）
3. Beta 不稳定性（随时间变化）

**实践中**：

- 使用 **Fama-French 三因子/五因子模型** 代替 CAPM
- 使用 **滚动窗口** 计算 Beta（捕捉时变特征）
- 结合 **基本面分析**、**技术分析** 等，形成综合判断
