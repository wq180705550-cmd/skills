# 燃料油（FU高硫+LU低硫）完整分析框架 v2.14.0

> **核心定位**：FU=高硫380CST渣油、对标新加坡MOPS380；LU=低硫船燃VLSFO；**原油定底、裂解定弹性、新加坡定锚、国内保税政策定独立行情**；FU跟原油相关性85%~92%，供需失衡阶段相关性骤降、走独立行情。

---

## 一句话精简复盘口诀

> **原油定底、裂解定强弱、新加坡定锚、保税政策定异动，夏季发电多FU、船旺低硫多LU，库存垒库全做空、去库基差走强做多。**

---

## 整体架构

**成本底座 → 裂解价差（核心强弱中枢） → 外盘锚（新加坡） → 国内供需+政策 → 结构（基差/月差/跨品种价差）**，宏观地缘、季节作为外部修正项。

> **定位**：原油决定大方向（涨跌天花板/地板），**裂解决定相对强弱（跑赢/跑输原油），供需与政策决定独立行情**。

---

## 一、第一层：成本驱动（行情大方向，价格解释力约60%；在量化打分模型中占25%顶层权重）

燃料油是原油蒸馏尾渣，**布伦特/SC原油是成本底线**：

### 1. 成本公式（保税净价，FU/LU都是保税交割）

```
进口完税成本 = 新加坡MOPS + 海运费 + 保险 + 升贴水 - 出口退税（13%船燃退税）+ 汇率损耗
```

> 美元走强→人民币贬值→同等MOPS，国内FU被动涨价；美元跌、人民币升值，进口成本下移。

### 2. 中东地缘传导（对标SC逻辑，但燃料油敏感度更高）

中东是全球高硫燃料油核心出口国（伊拉克、科威特、伊朗），霍尔木兹/红海扰动：

- **短期**：原油大涨+燃料油船运受阻→MOPS暴涨→FU高开
- **中期**：油轮绕行好望角，VLCC运费大涨，亚太进口成本刚性抬升

**区别SC**：燃料油无国储抛储缓冲，地缘脉冲涨幅经常大于原油

### 3. 俄油制裁关键变量

俄罗斯是全球高硫燃料油第一大出口国，欧美制裁→俄燃料油贴水收缩、亚洲到货减量，直接推升新加坡库存、抬升MOPS，是近年FU重要扰动项。

---

## 二、第二层：裂解价差（燃料油自身基本面，决定强弱分化，最核心观察指标）

**裂解价差 = FU现货价 - 折算原油成本（美元/吨）**，衡量炼厂产渣油利润，是脱离原油涨跌的独立驱动：

### 1. FU（高硫380）裂解两大需求去向

| 需求类型 | 地区/用途 | 时间特征 | 裂解影响 |
|---------|----------|---------|---------|
| 南亚发电刚需 | 印度、巴基斯坦 | 每年5-9月夏季用电高峰 | 裂差走强、FU强于原油 |
| 国内地炼掺炼 | 山东地炼 | 全年动态 | 地炼利润好→采购高硫增多、FU走强；地炼亏损停工→FU需求崩塌 |

> **规律**：裂差＞历史高位→FU偏强，独立涨；裂差深度贴水→FU弱于原油，原油涨FU不涨

### 2. LU（低硫）裂解

需求全是**保税国际船用加注（IMO限硫令后刚需）**，炼厂优先产柴油汽油，低硫渣油产出少，**LU裂解长期显著高于FU、韧性更强**

### 3. 高低硫价差（LU-FU）

| 价差状态 | 含义 | 交易机会 |
|---------|------|---------|
| 价差走扩 | 低硫紧、高硫松（炼厂少产低硫、船用刚需稳、高硫俄油大量到货） | 多LU空FU |
| 价差收窄 | 脱硫船增多（烧高硫）、高硫发电旺季来临，高硫需求回暖 | 多FU空LU |

---

## 三、第三层：外盘锚定——新加坡MOPS与新加坡库存（FU定价锚）

全球燃料油集散中心=新加坡，**FU现货定价直接锚定普氏新加坡MOPS380**，FU≈MOPS+远东套利价差：

### 1. 新加坡库存（周度数据）

| 库存变化 | 市场影响 |
|---------|---------|
| 库存持续去库 | MOPS上行、FU走强 |
| 库存垒库 | 外盘承压、FU跟跌 |
| 浮仓激增 | 代表货源过剩，远期承压 |

### 2. 远东套利船货

中东/欧美往新加坡发运船货增多→新加坡到货上升、价格走弱；船货减少→外盘走强带动FU

## 四、第四层：国内基本面（FU独有变量，走出独立行情关键）

### 1. 供应端

| 供应来源 | 说明 | 影响 |
|---------|------|------|
| 国内炼厂产出 | 中石化/中石油/浙石化炼厂加工重质原油越多，副产渣油越多 | 加工轻质油则FU供应收紧 |
| 燃料油进口量 | 海关月度数据 | 高硫进口大增→垒库承压；进口锐减→现货偏紧 |
| 仓单（上期所FU仓单） | 仓单大量注册=现货充裕 | 仓单低位→近月逼仓上涨 |

### 2. 需求端（分三块）

| 需求类型 | 主力品种 | 说明 |
|---------|---------|------|
| 保税船燃 | LU（低硫主力） | 舟山、上海、青岛保税加注量，BDI上涨→加注需求上升 |
| 高硫发电 | FU（外需） | 南亚印度、巴基斯坦夏季发电刚需 |
| 地炼掺炼 | FU（内需） | 山东地炼加工利润决定采购意愿 |
| 内贸工业燃料 | - | 占比萎缩，影响很小 |

### 3. 国内保税政策（极强政策扰动）

| 政策 | 影响 |
|------|------|
| **13%出口退税** | 保税加注视同出口享退税，退税稳定=国内炼厂愿意生产船燃 |
| **自贸区保税混兑** | 舟山/海南允许高低硫保税油混兑出0.5%低硫→高硫需求被动抬升 |
| **进口关税/消费税** | 关税上调→进口成本抬升；消费税抵扣放开→地炼采购高硫意愿提升 |

## 五、第五层：季节性规律（全年行情日历，FU最清晰）

| 时间 | 驱动逻辑 | FU方向 |
|------|---------|-------|
| 5~9月 | 印度/巴基斯坦夏季用电高峰，高硫发电旺季，裂差修复 | 震荡偏强，易强于原油 |
| 10~次年4月 | 发电需求回落，俄高硫集中到港，地炼季节性检修 | 整体偏弱，易弱于原油 |
| 春节前后 | 国内外贸停航、船燃淡季；炼厂停工 | 季节性低位 |

## 六、第六层：基差+月差（盘面结构，判断强弱拐点）

### 1. 基差 = 现货 - 期货

| 基差结构 | 市场信号 |
|---------|---------|
| 正基差走强（现货升水） | 现货紧、去库、期货易上涨 |
| 深贴水 | 现货过剩、垒库、期货承压走弱 |

### 2. 月间价差（近月 - 远月）

| 月差结构 | 市场信号 |
|---------|---------|
| Back（近月＞远月） | 现货紧缺、近月强 |
| Contango（远月＞近月） | 货源充足、远期宽松 |

> **实操**：原油涨+正基差+Back+新加坡去库→多头行情；原油跌+深贴水+Contango+新加坡垒库→空头行情

|---------|------|------|

## 九、模块2：裂解价差因子量化指标明细（权重30%）

FU/LU分开量化，全部可日度/周度计算，**模块总权重30%**。


```
CrudeCost = Brt(美元/桶) × 7.33 × USDCNY

```

## 十一、模块3：新加坡外盘因子（权重20%）

新加坡MOPS是FU现货定价锚，外盘决定内盘方向。

### 11.1 新加坡燃料油库存（模块内权重10%）

**原始数据源**：新加坡IES周度库存（普氏/新加坡交易所），每周五公布：
- 陆上储罐库存：高硫380+低硫VLSFO燃料油
- 海上浮仓库存：新加坡锚地停泊待卸燃料油船货

**合成公式**：
```
TotalStock = 岸罐库存 + 海上浮仓库存 (单位：千吨)
```

#### 三个量化派生指标（全部用来打分）

| 指标 | 计算公式 | 权重 | 计分规则 |
|------|---------|------|---------|
| 绝对库存分位 | `100 - (Stock-Min)/(Max-Min) × 100` | 6% | 反向：库存越高得分越低 |
| 库存周环比 | `(本周-上周)/上周` | 2% | 反向：环比大增→低分利空 |
| 库存同比 | `(本周-去年同周)/去年同周` | 2% | 反向：同比走高→低分利空 |

**综合库存得分**：
```
StockScore = RawScore×0.6 + ChgScore×0.2 + YoYScore×0.2
```

**信号阈值**：
- StockScore > 65：库存低位去库 → 利多FU
- StockScore < 35：库存高位垒库 → 利空FU

### 11.2 到港船货预估（模块内权重5%）

| 指标 | 数据频率 | 计算方式 | 计分规则 |
|------|---------|---------|---------|
| 未来4周亚太到港量同比 | 周度 | 同比变化 | **反向**：到货大增→得分降低 |

### 11.3 MOPS现货价格（模块内权重5%）

| 指标 | 数据频率 | 计算方式 | 计分规则 |
|------|---------|---------|---------|
| MOPS 380/VLSFO现货20日涨跌 | 周度 | 20日涨跌幅分位 | 正向：外盘涨价→利多内盘FU |

**模块3综合公式**：
```
SGScore = StockScore×0.5 + ArrivalScore×0.25 + MOPSScore×0.25
```

### 15.5 标准化得分函数

```python
def standard_score(x, window=365*3, is_forward=True):
    """
    x: 指标序列
    window: 滚动窗口，默认3年
    is_forward: 正向指标=True，反向指标=False
    """
    roll_min = x.rolling(window=window, min_periods=30).min()
    roll_max = x.rolling(window=window, min_periods=30).max()
    score = (x - roll_min) / (roll_max - roll_min) * 100
    if not is_forward:
        score = 100 - score
    return score.clip(0, 100)
```
