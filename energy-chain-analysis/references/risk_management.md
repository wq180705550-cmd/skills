# 燃料油风险管理体系

## 一、核心风险识别

### 大类风险与识别信号

| 风险类型 | 识别依据（分析体系指标） | 价格影响 | 风险等级 |
| ---- | ---- | ---- | ---- |
| **成本端风险** | 布伦特原油趋势突变、原油库存/炼厂数据大幅偏离预期、地缘冲突 | 全品种涨跌停、跳空缺口 | 极高 |
| **供需风险** | 新加坡/舟山库存骤增/骤降、BDI指数断崖式下跌、船燃需求骤变 | 趋势反转、价差快速走扩/收窄 | 高 |
| **政策风险** | IMO新规落地、关税/出口退税调整、保税规则变更、期货风控参数修改 | 结构性行情扭转、高低硫价差异动 | 极高 |
| **价差风险** | 内外价差大幅偏离合理区间、裂解价差/高低硫价差突破历史极值 | 套利头寸亏损、单边行情失真 | 中高 |
| **汇率风险** | 美元兑人民币单日大幅波动 | 内外价差异动、国内价格独立行情 | 中 |
| **资金&流动性风险** | 持仓/成交量骤降、主力合约移仓、交易所限仓 | 无法正常开平仓、滑点放大 | 中 |

### 风险预警阈值（直接盯盘使用）

1. **原油**：单日涨跌幅 > 3%，视为成本端风险预警，立即减仓避险
2. **内外价差**：> 60元/吨 或 < 0元/吨，价差回归风险，不追单边
3. **高低硫价差**：突破近1年80%分位，结构反转风险，套利头寸减仓
4. **库存**：单周库存变动 > 10%，供需拐点风险，调整持仓方向
5. **BDI指数**：单周跌幅 > 15%，航运需求骤降风险
6. **突发政策/地缘**：第一时间启动观望模式

## 二、通用基础风控规则（全场景强制执行）

### 1. 仓位管理（第一道防线）

燃料油波动率高于普通能化品，严格分级控仓：

| 场景 | 仓位上限 | 说明 |
|------|---------|------|
| 总账户上限 | 燃料油FU+LU合计 ≤ 总资金20% | 不可突破 |
| 单边投机 | 单笔品种 ≤ 总资金12% | 趋势明朗+多重共振最高不超15% |
| 价差套利 | 组合仓位 ≤ 总资金18% | 对冲后风险更低 |
| 极端行情预警 | 统一降至5%以内 | 原油大幅波动/政策落地前 |

### 2. 止损规则（基本面+技术面双保险）

摒弃纯技术止损，用分析体系指标划定止损边界：

#### 单边头寸止损
**趋势单（中线）**：
- 核心止损条件：布伦特原油趋势破位 + 新加坡MOPS同步走弱（多头）/走强（空头）
- 价格止损：关键高低点/中长期均线
- 亏损上限：单笔亏损严格控制在总资金3%以内

**波段单（短线）**：
- 核心止损条件：内外价差回归合理区间、库存数据反向
- 价格止损：区间上下沿
- 亏损上限：单笔亏损 ≤ 总资金2%

**原则**：基本面逻辑失效，无条件离场，不止损硬扛

#### 套利头寸止损
套利以价差均值回归为核心，不止损单边价格：
- 高低硫套利：价差突破历史90%分位且连续2日不回归 → 止损离场
- 裂解套利：裂差持续偏离合理区间、炼厂逻辑反转 → 止损
- 跨市套利：汇率/关税政策突变 → 立即平仓

### 3. 止盈与分批离场（锁定利润）

**单边趋势**：
- 第一止盈：内外价差回归20-40元合理区间，减仓50%
- 最终止盈：原油趋势走弱、库存由降转增，全部离场

**套利头寸**：
- 价差回到历史中枢区间，直接全部平仓，不博取极致行情

### 4. 隔夜&节假日风控（防范跳空风险）

燃料油受境外原油、新加坡现货、国际地缘影响极大：
- 国内长假、境外重要假期、地缘局势紧张期：**大幅降仓，仅保留底仓**
- 重大数据（原油库存、IMO会议、关税公告）公布前夜：不新开重仓头寸
- 周五收盘前：检查所有头寸是否需要降仓

## 三、分交易场景针对性风险管理

### 场景1：单边趋势交易（FU/LU 多/空）

**核心风险**：原油趋势反转、区域供需错配、政策突袭

**风控措施**：
1. 持仓逻辑校验：每日复盘核心指标（原油趋势、MOPS、库存、BDI），只要一项走坏即减仓
2. 对冲辅助：持仓较重时，小仓位反向搭配原油合约，对冲原油突发波动
3. 禁忌：原油震荡期、价差严重偏离时，禁止重仓追涨杀跌

### 场景2：价差套利交易

#### 高低硫套利（LU-FU）
**核心风险**：IMO政策调整、脱硫塔普及率突变、高硫发电需求暴增/锐减

**风控措施**：
- 政策窗口期暂停新开套利单
- 价差偏离极值后不再加仓摊薄成本
- 套利组合始终保持对等手数，杜绝单边敞口

#### 裂解价差套利（燃油↔原油）
**核心风险**：炼厂检修、原油加工结构变化

**风控措施**：
- 炼厂集中检修季，缩小套利仓位
- 裂差进入历史极端区间，停止加仓

#### 内外跨市套利（机构专属）
**核心风险**：汇率、关税、跨境物流、交割规则变化

**风控措施**：
- 汇率单日波动超1%，强制减仓
- 政策传闻阶段只观望不入场

### 场景3：产业套期保值

**核心风险**：套保比例失衡、基差/价差偏离、套保时机错误

**风控措施**：
1. 比例风控：根据库存、货量、船期动态调整套保比例，不超额套保、不投机加码
2. 基差风控：依托内外价差、现货期货基差，基差大幅走弱时分批调整
3. 政策风控：关税、出口退税调整前提前对冲

### 场景4：短线波段/日内交易

**核心风险**：短期情绪、滑点、无序波动

**风控措施**：
1. 仅用分析体系定大方向，绝不逆势
2. 单笔仓位 ≤ 总资金5%，严格小止损
3. 单日累计亏损达总资金4%，停止当日交易
4. 不持仓隔夜：规避境外盘突发波动

## 四、专项风险对冲工具

### 1. 对冲原油风险
- 燃料油头寸搭配原油期货（内外盘均可）
- 构建组合，抵消原油单边波动，保留燃油自身供需收益

### 2. 对冲汇率风险
- 内外价差交易、跨境贸易套保场景下
- 搭配外汇工具对冲人民币汇率波动

### 3. 对冲政策结构风险
- 同时持有FU+LU对等头寸
- 抵消IMO、限硫政策带来的高低硫结构冲击

## 五、黑天鹅&突发风险应急处置流程

### 第一阶段：信号触发
- 原油跳空、政策突发、涨跌停封板
- **动作**：立即停止新开仓

### 第二阶段：持仓处置
- 有盈利头寸：分批止盈离场，落袋为安
- 亏损头寸：次日开盘无条件平仓，不扛单、不锁仓

### 第三阶段：后市观望
- 行情企稳、基本面指标重新回归正常逻辑前，空仓观望
- 不主观预判拐点

### 第四阶段：复盘
- 记录风险触发点
- 更新预警阈值
- 完善应急流程

## 六、风控复盘与体系迭代

### 每日/每周复盘内容

1. **亏损来源分析**：
   - 是原油判断失误？
   - 供需数据错判？
   - 还是忽略政策风险？

2. **风控执行检查**：
   - 是否严格遵守仓位规则？
   - 止损是否按计划执行？
   - 有无侥幸心理持仓？

3. **指标有效性评估**：
   - 原有预警阈值是否需要调整？
   - 市场结构是否发生变化？

### 迭代优化方向
- 航运旺季：库存波动加剧，适当收窄库存变动预警阈值
- 政策密集期：进一步降低整体仓位上限
- 波动率放大期：收窄止损幅度，加快止盈节奏

---

## 七、极简风控执行清单（每日对照）

1. □ 核查原油、库存、价差、政策四大类风险预警信号
2. □ 总仓位、单品种仓位控制在规则以内
3. □ 所有头寸提前设置基本面+价格双重止损
4. □ 长假/数据/地缘敏感期完成降仓
5. □ 套利头寸核对手数对等，无单边敞口
6. □ 单日亏损触及阈值，停止交易
7. □ 突发黑天鹅，执行应急平仓+观望

## 八、风险管理量化标准汇总

| 指标 | 标准 | 动作 |
|------|------|------|
| 单笔最大亏损 | 账户3% | 触及立即止损 |
| 单日最大亏损 | 账户4% | 停止当日交易 |
| 周度最大亏损 | 账户8% | 降低仓位50% |
| 连续亏损3次 | - | 强制休息1个交易日 |
| 原油单日涨跌>3% | - | 减仓至5%以内 |
| 内外价差>60或<0 | - | 不追单边 |
| 库存单周变动>10% | - | 调整持仓方向 |
| BDI单周跌>15% | - | 降低仓位 |
| 高低硫价差>80%分位 | - | 套利头寸减仓 |
