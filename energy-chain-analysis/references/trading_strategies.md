# 燃料油三类交易策略详解

## 核心原则
根据七步决策流程的分析结论，选择对应的交易策略。策略选择逻辑：
- 原油趋势明确 → 单边趋势交易
- 原油无方向、高低硫价差/裂解价差有明显偏离 → 价差套利交易
- 原油震荡、价格区间明确 → 区间波段交易

---

## 策略一：单边趋势交易（最高适配，体系核心用法）

### 适用场景
- 原油有明确趋势方向（周线级别确认）
- 基本面供需有明显偏向
- 中线持仓（3~15个交易日）

### 多头策略（做多 FU/LU）

**入场条件（全部满足）**
1. 布伦特原油周线趋势向上，无明显见顶信号
2. 新加坡MOPS同步走强，新加坡/舟山库存低位或持续去库
3. 内外价差处于低位（<20元），进口亏损，国内供应支撑强
4. BDI指数上行，航运需求回暖
5. 高低硫价差走高 → 优先做多LU；价差收窄 → 优先做多FU

**加仓条件**
- 价格回调至MA20附近获得支撑
- 成交量在回调中萎缩
- 内外价差仍在低位

**减仓信号**
- 内外价差突破60元（升水过高）
- 库存由降转增
- BDI指数回落

**离场信号**
- 原油趋势破位（价格跌破MA60或连续3日收阴）
- 价格触及止损位
- RSI进入超买区（>70）且出现顶背离

### 空头策略（做空 FU/LU）

**入场条件（全部满足）**
1. 布伦特原油趋势向下（周线空头排列）
2. 两地库存持续累库，船燃需求走弱（BDI回落）
3. 内外价差大幅升水（>60元），套利抛压大
4. 出口退税/进口关税下调，国内供应增加

**禁忌**
- 地缘冲突期间不盲目做空
- IMO新政落地前夕不做空（防范突发跳涨）
- 原油趋势不确定时不做空

### 中长线趋势交易（持仓1周~数月）
- 适配人群：趋势交易者、产业套保盘、中大型资金
- 周期：周线/日线级别
- 体系用法：以原油大趋势定方向，叠加IMO政策、炼厂产能、航运景气、库存周期判断趋势延续性；用裂解价差、高低硫价差确认品种强弱

### 短线波段交易（持仓1~5个交易日）
- 适配人群：日内波段选手、普通投机散户
- 周期：日线/60分钟线
- 体系用法：依托内外价差、区域库存、汇率找入场拐点；原油定边界，结合盘面量能、资金情绪择时

## 策略二：价差套利交易（低风险，适合稳健型）

### 子策略1：高低硫套利（LU - FU 价差）

#### 正向套利（多LU、空FU）
- **适用条件**：
  - 限硫政策收紧、航运旺季
  - 脱硫塔普及率低
  - 价差处于历史低位（低于30%分位）
- **入场时机**：价差连续5日同方向变动确认趋势
- **止损**：价差突破历史90%分位且连续2日不回归 → 止损离场
- **止盈**：价差回到常规区间（300~500元/吨）全部平仓
- **仓位**：套利组合始终保持对等手数，杜绝单边敞口

#### 反向套利（多FU、空LU）
- **适用条件**：
  - 高硫进入发电旺季（中东夏季5-9月）
  - 低硫产能放量
  - 脱硫塔使用增加
  - 价差处于历史高位（高于80%分位）
- **止损/止盈**：同上逻辑反向

### 子策略2：跨品种套利（燃油 vs 原油，裂解价差）

#### 多燃油 + 空原油
- **适用条件**：裂解价差处于历史低位，预期燃油补涨
- **核心依据**：炼厂利润、原油加工结构
- **止损**：裂差持续偏离合理区间、炼厂逻辑反转

#### 空燃油 + 多原油
- **适用条件**：裂解价差处于历史高位，预期燃油走弱
- **止损**：同上逻辑反向

### 子策略3：内外盘跨市套利（上海 ↔ 新加坡）
- **适配人群**：大型贸易商、跨境机构、具备境外交易资质主体
- **周期**：短中线（3~10日）
- **核心依靠**：内外价差、汇率、进出口成本
- **风险**：汇率、关税、跨境物流、交割规则变化
- **个人投资者不建议实操**，仅可将价差作为单边交易参考

---

## 策略三：区间波段交易（原油震荡行情专用）

### 适用条件
- 原油无明确趋势（均线缠绕、区间反复）
- 价格有明确的上下轨

### 操作方法
1. **划定价格上下轨**：取近1~2个月高低点
2. **做多信号**：价格触及下轨 + 低库存 + 内外价差偏低（<20元）
3. **做空信号**：价格触及上轨 + 高库存 + 内外升水偏高（>60元）
4. **止损**：区间突破即止损（突破上轨则空单止损，突破下轨则多单止损）
5. **止盈**：目标区间另一端减仓80%，剩余20%趋势跟踪

### 注意事项
- 严格执行区间止损，不恋战
- 快进快出，持仓不超过3天
- 单笔仓位≤总资金8%

---

## 策略选择决策树

`
原油趋势？
├── 明确向上/向下 → 单边趋势交易
│   ├── 高低硫价差走高 → 选LU
│   └── 高低硫价差走低 → 选FU
├── 无方向、震荡 → 区间波段交易
│   └── 但价差有明显偏离 → 价差套利交易
└── 不确定 → 观望
`

## 适用场景总览

| 交易场景 | 持仓周期 | 核心使用体系模块 | 适配度 |
|---------|----------|------------------|--------|
| 长线趋势投资 | 1个月以上 | 原油趋势+IMO/国内政策+产能结构 | ★★★★★ |
| 中线波段单边 | 1~15日 | 原油+库存+BDI+内外价差 | ★★★★★ |
| 高低硫价差套利 | 3~20日 | 高低硫价差+供需结构+政策 | ★★★★★ |
| 燃油-原油裂解套利 | 5~30日 | 裂解价差+炼厂利润+原油趋势 | ★★★★☆ |
| 产业套期保值 | 随产销/航运周期 | 全体系 | ★★★★★ |
| 内外盘跨市套利 | 3~10日 | 内外价差+汇率+关税政策 | ★★★★☆（仅限机构） |
| 能源板块轮动配置 | 季度/半年 | 原油大趋势+长期政策格局 | ★★★☆☆ |

## 不适用场景

1. **纯日内超短线（炒单）**：体系只用来定大方向边界，入场离场以盘口技术为主
2. **纯技术博弈**：完全只看K线无视基本面，本体系无价值
3. **短期消息面炒作**：极端突发黑天鹅，价格由情绪主导
4. **高频量化算法交易**：依赖tick数据与统计模型，人工分析体系无法匹配
5. **跨品类跨界交易**：逻辑完全不互通
