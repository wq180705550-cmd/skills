# 能源产业链分析技能 v2.17.0

专业能源产业链（SC原油/BU沥青/FU高硫燃料油/LU低硫燃料油/PG液化石油气）分析决策技能，基于TqSdk实盘数据，完整数据流水线：

```
TqSdk 实盘行情 → DuckDB 存储 → Ta-Lib 技术指标 → 量化打分 → 策略型交易建议 → 分析报告
```

## 核心框架

**两层产业链结构**：
- 上游核心（原油）—— 权重40%，永远分析第一顺位
- 下游炼化产品（并列）—— 权重60%（沥青/FU/LU各20%）
- LPG独立模块——不参与顶层加权，拥有独立五层框架

**燃料油五层量化打分**：
- 成本因子(25%) + 裂解因子(30%) + 新加坡外盘(20%) + 国内供需(15%) + 盘面结构(10%)
- 22个可量化指标，模块内+顶层双层加权，标准化0~100分

**沥青六层量化打分**：
- 成本(40%) + 供应(20%) + 需求(25%) + 库存(10%) + 政策(5%) + 结构(0%)

**LPG五层量化打分**：
- 外部市场(25%) + 供应(20%) + 需求(30%) + 库存(15%) + 政策(10%)

## 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 配置环境变量

```bash
# TqSdk 认证
export TQSDK_USERNAME=your_username
export TQSDK_PASSWORD=your_password
```

### 3. 运行完整流水线

```bash
python scripts/run_pipeline.py
```

### 4. 单独生成报告

```bash
python scripts/generate_reports.py --type all
```

### 5. 信号回测

```bash
# 使用demo数据回测（无需DuckDB）
python scripts/backtest_signals.py --all --demo

# 使用真实数据回测
python scripts/backtest_signals.py --symbol FU --days 756
```

### 6. 风险管理

```bash
# 计算止损/仓位/交易成本
python scripts/risk_manager.py --symbol FU --price 3000 --atr 50 --direction long --account 300000
```

### 7. LPG CP价格获取

```bash
# 手动输入CP价格
python scripts/fetch_cp_price.py --propane 520 --butane 480

# 自动获取最新CP价格
python scripts/fetch_cp_price.py --auto
```

## 核心功能

### 1. 量化打分

| 品种 | 打分框架 | 权重 | 信号阈值 |
|------|----------|------|----------|
| FU/LU | 五层量化 | 成本25%+裂解30%+新加坡20%+供需15%+结构10% | >65做多, <35做空 |
| BU | 六层量化 | 成本40%+供应20%+需求25%+库存10%+政策5% | >65做多, <35做空 |
| PG | 五层量化 | 外部25%+供应20%+需求30%+库存15%+政策10% | >65做多, <35做空 |

### 2. 信号回测

- 滚动计算技术指标和量化得分
- 统计胜率、盈亏比、最大回撤、夏普比率
- 季节性分析（各月份收益率/波动率/胜率）
- 支持demo数据回测（无需DuckDB）

### 3. 风险管理

- **止损计算**：ATR法（1.5倍ATR止损，3倍ATR止盈）
- **仓位管理**：基于账户规模（默认30万），单笔最大亏损2%本金
- **交易成本**：手续费万分之1.2 + 保证金12% + 滑点1跳
- **极端行情预警**：价格异动>5%、波动率突增2x、涨跌停检测
- **信号时效性**：3天有效期，超期自动标记

### 4. 套利策略

| 策略 | 入场条件 | 止损 | 最大持仓 |
|------|----------|------|----------|
| LU-FU高低硫价差 | Z>2.0或Z<-2.0 | 3% | 20天 |
| SC-BU裂差 | Z>2.0或Z<-2.0 | 3% | 15天 |
| FU裂解价差 | Z>1.5或Z<-1.5 | 4% | 25天 |
| 月间价差 | Z>2.0或Z<-2.0 | 2% | 10天 |

### 5. 策略型交易建议

分析完成后，按标准模板输出具体交易建议：

| 内容 | 说明 |
|------|------|
| 市场研判 | 原油趋势、传导系数、数据质量 |
| 品种交易建议 | 信号/入场/止损/止盈/仓位/风险收益比/持仓周期 |
| 套利策略建议 | 价差/Z分数/入场条件/止损/止盈/最大持仓 |
| 风控提示 | 最大亏损/仓位上限/极端预警/信号有效期 |
| 关键风险 | 3条核心风险提示 |
| 执行清单 | 4项执行检查项 |
