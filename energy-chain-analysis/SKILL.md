## 技能描述

以**资深能源分析师**身份，基于**原油→沥青/FU/LU两层产业链**框架，对原油系全产业链执行七步决策流程，输出每日分析报告、交互式决策建议、自动化定时推送。

**覆盖品种**：
- **原油**：SC原油（INE）、布伦特原油、WTI原油
- **沥青**：BU主力（上期所沥青期货）
- **燃料油**：FU主力（高硫）、LU主力（低硫）

**核心框架（两层产业链结构）**：
1. **上游核心（原油）**——权重40%，永远分析第一顺位，决定产业方向
2. **下游炼化产品（并列关系）**——总权重60%，都直接来源于原油炼化
   - **沥青（BU）**——权重20%，道路建设需求
   - **高硫燃料油（FU）**——权重20%，对标新加坡MOPS380
   - **低硫燃料油（LU）**——权重20%，IMO新规清洁船燃VLSFO

**燃料油核心定位**：
- ⭐ **原油定底**：布伦特/SC原油是成本底线，决定价格上下边界
- ⭐ **裂解定弹性**：裂解价差决定FU/LU相对原油的强弱分化
- ⭐ **新加坡定锚**：普氏新加坡MOPS380是FU现货定价基准
- ⭐ **政策定异动**：国内保税政策（退税/混兑）决定独立行情

**LPG（液化石油气）核心定位**：
- ⭐ **CP定锚**：沙特丙烷合同价（CP）是全球LPG定价基准
- ⭐ **PDH利润定需求**：丙烷脱氢（PDH）利润决定化工需求强弱
- ⭐ **民用季节定节奏**：冬季取暖需求是民用消费核心驱动
- ⭐ **独立模块**：LPG不参与原油系顶层加权，拥有独立的五层分析框架

**核心铁律（2.4.0版更新）**：
- ⭐ **原油决定生死**：原油权重40%，永远是第一个分析的对象
- ⭐ **下游品种并列**：沥青、FU、LU都是原油直接下游产品，不是上下游关系
- ⭐ **炼化利润核心**：原油与各下游产品的价差反映不同炼化路线的利润
- ⭐ **产业链共振**：原油变动带动所有下游品种联动
- ⭐ **FU跟原油相关性85%~92%**，供需失衡阶段相关性骤降、走独立行情
- ⭐ **LPG独立分析**：PG不参与顶层加权，拥有独立的五层框架（成本25%+供应20%+需求30%+库存15%+政策10%）

### 数据源优先级策略（五层降级）

本技能采用**多源数据容错策略**，按优先级使用以下方案执行：

**Tier 0（最高优先级）：TqSdk 实盘行情**
- 使用 TqSdk 获取 SC/BU/FU/LU/PG 期货行情和K线数据
- 合约符号：
  - `KQ.m@INE.sc` — SC原油主力
  - `KQ.m@SHFE.bu` — BU沥青主力
  - `KQ.m@SHFE.fu` — FU高硫燃料油
  - `KQ.m@INE.lu` — LU低硫燃料油
  - `KQ.m@DCE.pg` — PG液化石油气（大连商品交易所）
- 认证：环境变量 `TQSDK_USERNAME` 和 `TQSDK_PASSWORD`
- 可用脚本：`scripts/fetch_and_store.py`

**Tier 1（第一优先级）：专业API数据源**
1. `neodata-financial-search` - 专业金融数据（SC/BU/FU/LU/PG）
2. `westockdata` - 技术面分析数据

**Tier 2（第二优先级）：WebSearch + WebFetch**
3. 通用网络搜索 + 权威媒体网站抓取
4. **LPG专项**：同花顺/东方财富月度CP价格报道

**Tier 3（第三优先级）：中文财经媒体**
5. 新浪财经、东方财富、财联社、隆众资讯（FU/LU/BU）
6. **LPG专项**：卓创资讯、百川盈孚（LPG）

**Tier 4（终极保底）：AI推理+历史推演**
7. 基于历史数据 + 合理推演（仅在前四方案全部失败时使用，必须标注"数据暂缺，基于历史数据推演"）

### COT数据源（CFTC交易者持仓报告）

COT原始数据**唯一官方源头是美国CFTC**，每周五15:30 ET发布，统计当周二持仓，存在3天滞后。能源产业链分析重点关注**Disaggregated报告**中的Managed Money净多头和Producer净空头。

**官方免费**：
- CFTC官网：https://www.cftc.gov/MarketReports/CommitmentsofTraders/index.htm
- CFTC Socrata API：`https://publicreporting.cftc.gov/resource/6dca-aqww.json`（无密钥，JSON输出）
- 历史批量包：`https://www.cftc.gov/files/dea/history/fut_disagg_txt_YYYY.zip`

**国内免费**：金十数据（全中文、自动净持仓）、文华财经/博易大师（外盘板块内置COT图表）

**海外免费**：TradingView（社区COT指标叠加K线）、Barchart.com（52周极值区间）

**付费专业**：Sentimentrader（极值预警）、Bloomberg（`CTOC`函数）、Quandl（Nasdaq Data Link）

**报告类型**：Legacy（3类快速判断）、Disaggregated（5类商品深度分析，**能源品种用此报告**）、TFF（5类金融期货）

**关键提醒**：COT滞后3天，短线参考有限；量化回测优先CFTC官方API

## 执行协议（必须遵守）

### 阶段0：原油前置分析（强制第一位）
1. **永远先分析原油**：SC原油永远是分析第一顺位
2. **确认原油趋势**：多头/空头/震荡
3. **计算原油传导系数**：原油信号强度对下游产品的影响系数

### 阶段1：数据准备（并行执行）
1. 并行获取行情数据（TqSdk优先 → neodata → WebSearch）
2. 并行获取新闻事件（WebSearch）
3. 并行获取技术指标（Ta-Lib计算）
4. 所有数据返回后，进入阶段2

### 阶段2：数据整合与质量检查
1. 验证所有数据时效性（48小时内）
2. 验证至少1个权威来源
3. 进行数据质量评级（高/中/低）
4. 缺失数据标注"数据暂缺"

### 阶段3：七步决策分析（两层产业链）
1. **原油趋势判断**：SC原油定方向（权重40%）
2. **沥青传导分析**：BU沥青作为中间体（权重20%），连接原油与燃料油
3. 品种强弱判断：高低硫价差，选FU/LU
4. 裂解价差分析：原油-燃料油成本利润率
5. 原油-沥青裂差分析：炼化利润评估
6. 政策与地缘风险排查
7. 风控与离场规则

### 阶段4：内容撰写
1. **第一章节永远是原油核心分析**（SC原油置顶）
2. **第二章节为沥青传导分析**（BU沥青中间体）
3. 后续章节跟进FU/LU分析
4. 确保所有铁律遵守

### 阶段5：最终审核
1. 检查原油分析是否在首位
2. 检查沥青传导分析是否完整
3. 检查权重计算是否正确
4. 验证所有关键约束遵守

## 核心分析框架：七步决策流程

### 第一步：判断整体趋势（原油定方向 - 权重40%）

**核心观察标的：SC原油主力 + 布伦特原油**

#### 1.1 原油多维度分析
1. **均线系统**：MA5、MA10、MA20、MA60排列
2. **RSI(14)**：超买超卖与背离
3. **MACD**：金叉死叉、柱体强度、背离
4. **量仓关系**：持仓量变化与价格配合
5. **高低点**：是否创新高/新低

#### 1.2 原油趋势结论
- **强多头**：MA多头排列 + RSI>50 + MACD金叉/红柱扩张
- **偏多头**：MA偏多 + RSI40~60
- **震荡**：均线交织 + RSI30~70 + MACD红绿柱交替
- **偏空头**：MA偏空 + RSI40~60
- **强空头**：MA空头排列 + RSI<50 + MACD死叉/绿柱扩张

#### 1.3 原油传导系数
- **强趋势**：原油信号传导系数 0.8-1.0（沥青/燃料油高度跟从）
- **弱趋势**：传导系数 0.5-0.7（沥青/燃料油部分跟从）
- **震荡**：传导系数 0.3-0.4（更多关注区域基本面）

---

### 第二步：下游炼化产品分析（并列关系 - 各20%权重）

**核心观察标的：BU沥青主力 + FU主力 + LU主力**

#### 2.1 品种基本面分别分析

**沥青（BU）分析**：
- **道路需求旺季**（3-6月、9-11月）：沥青需求旺盛
- **道路需求淡季**（1-2月、7-8月）：沥青需求疲软
- **库存变化**：厂库库存与社会库存趋势

**沥青季节性特征**：
| 月份 | 特征 | 策略参考 |
|------|------|---------|
| 3-6月 | 道路建设旺季 | 需求支撑，偏多 |
| 7-8月 | 高温雨季，需求淡季 | 震荡或偏空 |
| 9-11月 | 赶工期，需求旺季 | 需求支撑，偏多 |
| 12-2月 | 冬季停工，淡季 | 需求疲软 |

**燃料油（FU/LU）分析**：
- **高低硫价差（LU - FU）**：
  - 价差 > 500元/吨：LU偏强，可考虑多LU空FU套利
  - 价差 300~500元/吨：正常区间，根据其他信号选择
  - 价差 < 300元/吨：FU偏强，LU被低估，可考虑多FU空LU套利

#### 2.2 炼化价差分析（各品种与原油价差）

**原油-沥青裂差**：反映沥青炼化路线利润
- **裂差扩大**：沥青加工利润好，炼厂增产沥青
- **裂差收窄**：沥青加工利润差，炼厂减产或转产

**原油-燃料油裂解价差**：反映燃料油炼化路线利润
- **裂解价差扩大**：炼油利润好，生产积极性高，供应充足，利空燃料油
- **裂解价差收窄**：炼油利润差，生产积极性低，供应偏紧，利多燃料油

---

### 第三步：品种强弱对比分析（权重10%）

比较 BU、FU、LU 三个品种的相对强弱：
- 观察各品种与原油的相关性
- 比较各品种的技术面走势
- 分析各品种的供需面差异
- 选择相对强势或符合趋势的品种进行交易

---

### 第四步：政策与地缘风险排查（权重5%）

1. **IMO新规影响**：限硫、碳强度（影响燃料油需求结构）
2. **关税政策**：燃料油/沥青进口关税、出口退税
3. **沥青消费税**：财政部政策调整（2020年已调整）
4. **地缘政治**：中东局势、马六甲海峡状况（影响原油成本）
5. **库存周期**：连续4周以上同方向变动 = 趋势强化

---

### 第五步：盘面资金与技术择时（权重5%）

1. **持仓量变化**：持仓大增+放量上涨 = 多头确认，持仓大增+放量下跌 = 空头确认
2. **技术指标共振**：3个以上指标同向 = 信号增强
3. **跨品种联动**：原油带动所有下游品种联动

---

### 第六步：风控与离场规则（权重5%）

#### 仓位规则
- 单边交易：单品种仓位 ≤ 总资金12%（趋势明朗最高不超15%）
- 套利交易：组合仓位 ≤ 总资金18%
- 极端行情预警：降至5%以内或空仓

#### 止损规则
- 单边交易：ATR(14)×1.5 或 2% 本金，取小
- 套利交易：ATR(14)×0.8 或 1% 本金，取小
- 止损必须严格执行，禁止扛单

#### 止盈规则
- 分批次止盈：第1目标位止盈30%，第2目标位止盈30%，剩余40%持有并设置移动止损

## 三类交易策略

### 策略一：单边趋势策略（原油驱动型）
- **入场条件**：原油强趋势 + 沥青/燃料油同向共振 + 高低硫价差配合
- **止损设置**：ATR×1.5
- **止盈设置**：ATR×3（第一目标），ATR×6（第二目标）
- **仓位**：8%~12%

### 策略二：产业链套利策略
- **原油-沥青裂差套利**：原油/沥青比价偏离历史均值
- **沥青-燃料油套利**：BU-FU/BU-LU价差偏离正常区间
- **高低硫套利**（LU-FU）：正套/反套
- **入场条件**：价差偏离正常区间
- **止损设置**：价差继续偏离50元/吨
- **止盈设置**：回归到正常区间（或回归到均值）
- **仓位**：15%~18%

### 策略三：区间波段策略
- **入场条件**：原油震荡 + 沥青/燃料油在20日区间内
- **止损设置**：区间突破2%
- **止盈设置**：区间高低点附近
- **仓位**：5%~8%

### 策略四：季节性策略（沥青专用）
- **旺季做多**：3-6月、9-11月，道路建设需求旺季
- **淡季做空**：12-2月，冬季停工淡季
- **注意**：需结合原油趋势，逆势季节性交易需谨慎

## S_appendix：技能附录（重要约束与常见失误）

> **重要提示**：本附录包含关键约束和常见失误，使用本技能时必须严格遵守

### 【必须执行】关键约束

1. **原油永远是第一顺位**：无论任何分析场景，必须先分析原油（SC权重40%），再分析下游产品
2. **下游品种并列关系**：沥青、FU、LU都是原油直接下游产品，不是上下游关系，不可用沥青去推导原油
3. **LPG独立模块**：PG不参与原油系顶层加权，必须单独分析，不可与FU/LU混淆
4. **数据时效性**：所有数据必须验证时效性（48小时内），标注"数据暂缺，基于历史数据推演"
5. **DuckDB优先存储**：所有数据必须通过DuckDB存储，禁止使用CSV中间文件

### 【常见失误】执行失误警示

#### ❌ 失误1：先分析下游再分析原油
**后果**：违背"原油决定生死"铁律，分析方向可能完全错误
**修正**：强制要求第一章节必须是原油分析

#### ❌ 失误2：混淆FU/LU与PG分析框架
**后果**：FU/LU使用五层框架（成本+裂解+新加坡+国内供需+盘面），PG使用五层框架（外部市场+供应+需求+库存+政策），混用导致打分失真
**修正**：PG有独立的LPGScorer，不可混用CompleteFuelOilScorer

#### ❌ 失误3：忽视季节性调整
**后果**：沥青冬季（12-2月）和LPG夏季（4-8月）是淡季，不调整会导致打分偏高；燃料油5-9月是印度发电旺季，FU需求走强
**修正**：BitumenScorer和LPGScorer会自动应用季节性调整，也可手动调用`calculate_seasonal_adjustment()`

#### ❌ 失误4：PG数据使用错误
**后果**：PG的CP丙烷合同价是美元计价，混淆汇率会导致成本计算错误
**修正**：CP价格需要乘以USDCNY汇率转换为人民币成本

#### ❌ 失误5：跳过数据验证
**后果**：使用过期或错误数据会导致完全错误的分析结论
**修正**：所有数据必须验证来源和时效性

### 【强调标记】关键警告

> ⚠️ **警告1**：本技能分析结果仅供参考，不构成投资建议。期货交易具有高风险。

> ⚠️ **警告2**：量化打分得分>65不代表一定会涨，<35不代表一定会跌，必须结合原油趋势判断。

> ⚠️ **警告3**：LPG作为独立模块，其分析结果不参与原油系顶层加权打分。

## 第零原则：右侧交易铁律（全局强制）

**所有量化打分和交易建议必须基于已确认的右侧价格行为信号，禁止左侧猜测。**

### 核心要点

1. **状态 ≠ 信号**：得分>65（多头区间）、RSI<30（超卖）、CCI<-100（超卖）等只描述当前市场处于什么状态，**不代表价格即将反转或延续**。高分≠立刻做多，低分≠立刻做空。

2. **强趋势中指标会"钝化"**：在强空头趋势中，RSI可以连续数周<30；在强多头趋势中，RSI可以连续数周>70。ADX>25确认强趋势存在时，价格沿趋势方向运动的概率远大于反转。

3. **量化评分是状态描述，不是交易信号**。打分系统的输出（>65多/35-65震荡/<35空）反映的是当前市场各因子的综合状态，不是入场信号。**必须等待右侧价格行为确认后，才能将评分结论转化为交易建议。**

4. **必须等待右侧确认**。右侧信号包括：
   - **MA确认突破/跌破**：价格站上/跌破MA20且MA20走平或拐头
   - **MACD确认**：MACD底背离/顶背离确认（至少出现2次金叉/死叉后的方向确认）
   - **K线形态确认**：反转形态（锤子线/吞没/双底/头肩等）在下一根K线被确认
   - **DMI确认**：+DI和-DI在ADX>25背景下完成交叉
   - **量仓配合**：放量+持仓增加验证价格突破有效性

5. **无信号则观望（强制）**：没有任何右侧信号被确认时，策略建议必须是"观望/等待"。禁止用"轻仓试多/试空"替代观望。

6. **套利策略的右侧触发**：套利（价差回归）策略也需要右侧确认——至少等待价差在阈值位置出现停顿或反转信号后再入场，禁止仅因Z分数达到阈值就机械开仓。
