# S_body：技能主体

## 技能描述

以**资深贵金属交易员和分析师**身份，提供**完整的贵金属交易决策分析辅助**。本系统将交易决策拆解为三层架构，每层独立运行但层层约束，防止周期混淆与载体错配。

**核心定位**：交易决策辅助系统（非每日资讯推送），专注于：
- **宏观决策层（Level 1）**：判定市场机制（Regime）与主方向
- **载体选择层（Level 2）**：针对同一方向选出最优表达工具
- **多周期执行层（Level 3）**：入场、止损、仓位管理的精细化

**适用品种**：黄金、白银、铂金、钯金（以黄金、白银为主）

**交易工具**：
- 国际现货/期货（XAU/USD、GC、SI）
- 国内现货/期货（沪金AU、沪银AG）
- 国内外ETF（GLD、SLV、IAU等）
- 纸黄金、期权（GC/GLD/沪金期权等）

**操作周期**：日线波段、4H/1H/15m全周期覆盖

---

## 三大铁律（不可违反）

1. **Level 1 拥有最高优先级**：任何更低周期的信号不得建立与Level 1方向相反的仓位。
2. **Level 2 必须确认载体与宏观锚的偏差**：同一方向，不同载体（如沪金vs国际金）可能存在汇率溢价或流动性差异，需每日核对。
3. **Level 3 必须遵守周期层级纪律**：日线裁判、4H定结构、1H/15m只做触发，禁止跨级越权。

## 第一章 核心定价驱动（四大因子）

所有分析、开仓、止损均围绕以下四层逻辑展开。权重排序随市场机制变化，但始终需逐一排查。

### 1.1 四大因子定义

| 因子 | 作用周期 | 核心指标 | 备注 |
|------|----------|----------|------|
| **实际利率** | 中长期决定性 | 10年期TIPS收益率、10年期名义收益率-CPI/PCE | 黄金0息资产，实际利率下行利多，上行利空。但2022年后央行购金使此关系部分松动。 |
| **美元指数** | 中短期强相关 | DXY、欧元/日元交叉盘 | 负相关为主，极端危机时可能同步上涨（避险买盘）。 |
| **避险情绪** | 脉冲式短线 | VIX、GVX、VXSLV、地缘事件、银行信用风险 | 来得快去得快，不适合长线持仓。 |
| **工业需求** | 白银/铂钯核心 | 光伏装机、汽车产销、催化剂用量 | 黄金工业占比<10%，白银>50%。经济复苏利多银，衰退利空银。 |

### 1.2 附加因子（2022年后重要性上升）

- **央行购金**：季度级慢变量，提供底部支撑，削弱实际利率的压制效力。
- **美元信用折价**：美债/GDP、财政赤字、去美元化进程。当此因子主导时，黄金与美元/实际利率可能同步走强。

### 1.3 波动率指数观测（GVX/VXSLV）- 动态参数调整

**CBOE黄金ETF波动率指数（GVX）**：
- 代码：GVX（CBOE Gold ETF Volatility Index）
- 本质：基于GLD期权的隐含波动率，类似VIX对S&P500
- **动态阈值调整机制**：
  - 基于90天滚动窗口计算分位数
  - 根据市场状态（R1-R5）动态调整阈值敏感度
  - R1（降息式多头）：阈值提高10%（更宽容）
  - R5（流动性危机）：阈值降低20%（更敏感）
- 历史分位参考（基于2020-2025年数据）：
  - 25%分位：13.5
  - 50%分位（中位数）：18.2
  - 75%分位：24.8
  - 90%分位：31.5
  - 95%分位：38.2
- 关键阈值（动态调整）：
  - GVX < 13.5（<25%分位）：极低波动，市场极度平静，买权成本最低
  - GVX 13.5-18.2（25%-50%分位）：低波动，适合趋势策略
  - GVX 18.2-24.8（50%-75%分位）：中等波动，正常状态
  - GVX 24.8-31.5（75%-90%分位）：高波动，不确定性上升，减仓
  - GVX > 31.5（>90%分位）：极高波动，危机模式，只平不开
  - GVX > 38.2（>95%分位）：极端波动，流动性危机警戒
- 信号解读：
  - GVX急升 + 金价上涨 → 避险需求主导，上涨可能持续
  - GVX急升 + 金价下跌 → 流动性危机，黄金被抛售换现金
  - GVX低迷 + 金价上涨 → 实际利率/美元驱动，趋势健康
  - GVX低迷 + 金价下跌 → 阴跌模式，无恐慌但持续走弱

**CBOE白银ETF波动率指数（VXSLV）**：
- 代码：VXSLV（CBOE Silver ETF Volatility Index）
- 本质：基于SLV期权的隐含波动率
- **动态阈值调整机制**：
  - 基于90天滚动窗口计算分位数
  - 根据市场状态（R1-R5）动态调整阈值敏感度
  - R1（降息式多头）：阈值提高10%（更宽容）
  - R5（流动性危机）：阈值降低20%（更敏感）
- 历史分位参考（基于2020-2025年数据）：
  - 25%分位：18.5
  - 50%分位（中位数）：26.3
  - 75%分位：35.8
  - 90%分位：45.2
  - 95%分位：52.5
- 关键阈值（动态调整）：
  - VXSLV < 18.5（<25%分位）：极低波动，买权成本最低
  - VXSLV 18.5-26.3（25%-50%分位）：低波动，适合趋势策略
  - VXSLV 26.3-35.8（50%-75%分位）：中等波动，正常状态
  - VXSLV 35.8-45.2（75%-90%分位）：高波动，减仓
  - VXSLV > 45.2（>90%分位）：极高波动，只平不开
  - VXSLV > 52.5（>95%分位）：极端波动，流动性危机警戒
- 信号解读：
  - VXSLV > GVX × 1.5 → 白银波动率溢价过高，可能回归
  - VXSLV急升 + 银价上涨 → 白银投机情绪高涨
  - VXSLV急升 + 银价下跌 → 白银去杠杆/恐慌抛售

**动态仓位调整机制**：
- 根据GVX水平动态调整仓位乘数：
  - 极低波动（<25%分位）：仓位乘数1.0
  - 低波动（25%-50%分位）：仓位乘数1.0
  - 中等波动（50%-75%分位）：仓位乘数1.0
  - 高波动（75%-90%分位）：仓位乘数0.5
  - 极高波动（>90%分位）：仓位乘数0.0

**动态止损调整机制**：
- 根据GVX水平动态调整止损乘数：
  - 极低波动（<25%分位）：止损乘数1.5（更宽松）
  - 低波动（25%-50%分位）：止损乘数1.5（更宽松）
  - 中等波动（50%-75%分位）：止损乘数1.0（标准）
  - 高波动（75%-90%分位）：止损乘数0.75（更紧凑）
  - 极高波动（>90%分位）：止损乘数0.5（非常紧凑）

**GVX/VXSLV组合信号矩阵**：

| GVX状态 | VXSLV状态 | 综合信号 | 交易含义 | 仓位调整 |
|---------|-----------|----------|----------|----------|
| 极低（<13.5） | 极低（<18.5） | 市场极度平静 | 趋势策略为主，买权成本最低 | 仓位100% |
| 低（13.5-18.2） | 低（18.5-26.3） | 低波动 | 趋势策略为主，仓位正常 | 仓位100% |
| 中（18.2-24.8） | 中（26.3-35.8） | 正常波动 | 标准仓位，注意止损 | 仓位100% |
| 高（24.8-31.5） | 高（35.8-45.2） | 高波动 | 减仓，收紧止损 | 仓位50% |
| 极高（>31.5） | 极高（>45.2） | 极高波动 | 只平不开，等待回归 | 仓位0% |
| 高（>24.8） | 低（<26.3） | 分化 | 黄金避险，白银平静，关注金银比 | 黄金仓位75%，白银仓位50% |
| 低（<18.2） | 高（>35.8） | 分化 | 白银异常，可能有独立事件 | 黄金仓位100%，白银仓位50% |
| 极高（>38.2） | 任何 | 极端黄金波动 | 流动性危机警戒，只平不开 | 仓位0% |
| 任何 | 极高（>52.5） | 极端白银波动 | 白银流动性危机，金银比策略暂停 | 白银仓位0% |

## 第二章 市场机制分类（Regime）与诊断方法

将市场划分为5种典型机制（Regime），每种机制对应特定的策略偏好与禁用清单。

### 2.1 五种Regime定义

| 编号 | 名称 | 典型特征 | 实际利率 | 美元 | 央行买盘 | 金银表现 |
|------|------|----------|----------|------|----------|----------|
| **R1** | 降息/衰退式多头 | CPI回落，降息预期发酵或已落地 | 下行趋势 | 走弱 | 托底 | 黄金稳健牛，白银弹性释放 |
| **R2** | 高利率震荡（横盘） | 利率高位横盘，官员分歧，无明确方向 | 高位横盘 | 偏强震荡 | 托而不举 | 大区间震荡，趋势不延续 |
| **R3** | 鹰派重定价（加息/更高-for-longer） | 通胀粘性，美联储暗示再加息或维持高利率 | 上行/高位 | 走强 | 抵抗但挡不住 | 承压下行，反弹即卖 |
| **R4** | 信用折价/去美元化溢价 | 地缘冲突、美债信用担忧、央行持续大额购金 | 常规或偏高 | 不一定强 | 强买 | 金价抗跌甚至逆势上涨，实际利率框架部分失效 |
| **R5** | 流动性危机清算模式 | 美元融资枯竭，SOFR飙升，跨货币基差走阔 | 不关键 | 美元融资紧张 | 无关 | 一切相关性→1，黄金被抛售换现金 |

### 2.2 每日Regime诊断清单（开盘前3分钟完成）

```
□ 10年期TIPS方向（上行/下行/横盘）
□ 2年期美债收益率方向（反映货币政策预期）
□ DXY最近5日走势 + 是否突破关键位（如100/105/110）
□ SOFR/美元交叉货币基差有无异常跳升？（有→立即进入R5警戒）
□ 金价与200日均线关系（之上/之下/附近）
□ CFTC非商业净多头变化（增加/减少/极端拥挤）
□ 本周重大事件（CPI/非农/FOMC）→ 若有，仓位上限减半
□ 央行购金月度数据（WGC）是否维持高位（>800吨/年节奏）
□ GVX当前水平及趋势（急升/平稳/低迷）
□ VXSLV当前水平及趋势（急升/平稳/低迷）

综合判断：勾选结果映射到R1-R5，并记录在交易日志中。
```

### 2.3 Regime诊断快速映射表

| 诊断结果 | Regime | 首选策略 | 禁用策略 |
|----------|--------|----------|----------|
| TIPS下行 + DXY走弱 + 央行托底 | R1 | 宏观利率波段做多、均线顺势 | 趋势做空 |
| TIPS横盘 + DXY震荡 + 无方向 | R2 | 区间高抛低吸、期权Iron Condor | 趋势追单 |
| TIPS上行 + DXY走强 + 鹰派发言 | R3 | 宏观利率波段做空、反弹做空 | 趋势做多、央行抄底 |
| 央行大额购金 + 地缘冲突 + 信用担忧 | R4 | 央行抄底长线多 | 卖Vol |
| SOFR飙升 + 基差走阔 + 流动性紧张 | R5 | 只平仓不开新仓 | 所有策略 |

## 第三章 策略库（按Regime分类）

每个策略均标注：适用Regime、首选载体、周期裁判、一票否决条件。

### 3.1 宏观利率波段策略（长线，持仓数周~数月）

| 项目 | 内容 |
|------|------|
| **适用Regime** | R1（做多）、R3（做空） |
| **首选载体** | GC期货 / GLD ETF / 沪金AU |
| **周期裁判** | 日线（确认趋势）+ 4H（寻找回踩位） |
| **入场条件** | 做多：实际利率持续下行+价格回踩60/200日均线+CFTC多头未极端拥挤；做空：实际利率上行+美元走强+反弹至均线压力 |
| **止损** | 趋势结构外侧（如波段低点/高点下方/上方） |
| **一票否决** | ① SOFR/basis异常→降仓或平仓；② 突发避险打乱利率主线→暂停加仓；③ 实际利率横盘→休眠 |

### 3.2 央行抄底长线多头策略

| 项目 | 内容 |
|------|------|
| **适用Regime** | R4（信用折价区）或R1中的急跌后 |
| **首选载体** | GLD/IAU（长线持有）、纸黄金（无杠杆）、远期期货（慎用杠杆） |
| **周期裁判** | 周线（判断价值带）+ 日线（确认急跌超5-8%） |
| **入场条件** | ① 金价短期急跌5%以上，无长期货币政策利空；② 央行月度购金维持高位；③ CFTC空头拥挤但中长期降息逻辑未破坏 |
| **止损** | 前一段下跌结构外部低点外侧 |
| **一票否决** | ① 美联储重启强力加息；② 央行数据转为大幅减持；③ 处于R3鹰派加速期 |

### 3.3 金银比均值回归套利策略

| 项目 | 内容 |
|------|------|
| **适用Regime** | R1过渡期、R2震荡期（宏观平稳） |
| **首选载体** | GC期货（空金腿）+ SI期货（多银腿），等值Notional对冲 |
| **周期裁判** | 日线（比价极值）+ 4H（确认动量衰减） |
| **开仓条件** | 比价≥95（保守≥100）且宏观不在美元流动性收缩状态，且银工业端无需求崩塌；比价<50不作为开空白银触发，仅作为减多/防轧空区 |
| **平仓** | 比价回到80-85附近分批平；若比价继续朝不利方向突破前高+宏观流动性偏紧→砍 |
| **一票否决** | ① 极端避险（黄金独涨，银不跟）；② 经济大衰退（工业需求崩塌）；③ 比价在50-85中性区间 |

### 3.4 均线顺势波段策略（20+60日均线，持仓1-10日）

| 项目 | 内容 |
|------|------|
| **适用Regime** | R1（多头趋势）、R3（空头趋势） |
| **首选载体** | GC / 沪金AU / AG（半仓） |
| **周期裁判** | 4H（确认均线排列）+ 日线（过滤大方向） |
| **入场条件** | 20日线上穿60日线→回踩低多；20日线下穿60日线→反弹高空 |
| **止损** | 均线下方/上方一定距离（或ATR倍数） |
| **一票否决** | ① 均线粘合缠绕（金叉死叉频繁）；② ATR(14)低于30%分位（低波动假趋势）；③ 重磅数据前1-2天 |

### 3.5 区间震荡高抛低吸策略

| 项目 | 内容 |
|------|------|
| **适用Regime** | R2（横盘震荡） |
| **首选载体** | 期货轻仓 / GLD Covered Call / 期权Iron Condor |
| **周期裁判** | 4H（界定箱体上下沿）+ 1H（触发挂单） |
| **入场条件** | 价格触及箱体下沿做多，上沿做空，严格止损于箱体外侧2-3美金 |
| **止损** | 箱体外侧+0.5ATR |
| **一票否决** | ① 本周有CPI/非农/FOMC；② 宏观逻辑即将切换（突破前夕）；③ 箱体宽度<3美金（利润不够滑点） |

### 3.6 突破追单策略

| 项目 | 内容 |
|------|------|
| **适用Regime** | R1明确趋势（向上突破）、R3（向下突破） |
| **首选载体** | 期货（速度快） |
| **周期裁判** | 4H（识别结构突破）+ 成交量/OI确认 |
| **入场条件** | 价格放量突破震荡区间上沿/下沿，且OI上升（新钱入场） |
| **止损** | 突破K线起点或前震荡区间内部 |
| **一票否决** | ① 突破缩量或OI下降（假突破）；② 窄幅微型震荡（<3美金） |

### 3.7 事件驱动避险短线策略

| 项目 | 内容 |
|------|------|
| **适用Regime** | 任何Regime（但需先检查流动性） |
| **首选载体** | 期货轻仓 / 买Call（有限风险） |
| **周期裁判** | 1H-15m（脉冲管理） |
| **入场条件** | 突发事件后第一波拉升不追，等待回调支撑轻仓多，情绪消退立刻止盈 |
| **止损** | 入场点下方1-2ATR |
| **一票否决** | ① VIX飙升但SOFR也飙升（流动性危机→黄金先被卖）；② 冲突快速缓和（1小时内消退）；③ 长期战争（避险逻辑被利率覆盖） |

### 3.8 逆势抄底/摸顶策略（高风险，仅限专业）

| 项目 | 内容 |
|------|------|
| **适用Regime** | Regime拐点（如R3→R1转换初期） |
| **首选载体** | 买Call/Put（有限亏损），禁止期货逆势重仓 |
| **周期裁判** | 日线结构带 + CFTC持仓极值 |
| **入场条件** | 同时满足：① 宏观拐点硬证据（点阵图下移/前端利率形成下降通道）；② CFTC非商业净多头进入历史<10%或>90%分位且已开始反转；③ 价格进入预设结构性支撑/阻力带 |
| **止损** | 反向突破结构立刻认错 |
| **一票否决** | 缺少任一条件→零仓；普通散户禁用 |

| **一票否决** | ① 极端避险（黄金独涨，铂不跟）；② 经济大衰退（工业需求崩塌）；③ 比价在0.9-1.1中性区间 |

## 第四章 载体选择指南

同一宏观方向，不同载体因汇率、杠杆、成本、流动性差异会产生不同结果。

### 4.1 各载体本质属性

| 载体 | 跟踪标的 | 杠杆 | 关键隐性成本 | 最佳用途 |
|------|----------|------|--------------|----------|
| XAU/USD（伦敦现货） | 实物现货基准 | 取决于经纪商 | 隔夜swap（常为负） | 短线参考锚 |
| GC（COMEX黄金期货） | 现货+carry | 高（保证金） | 滚续成本、保证金追缴 | 波段主力 |
| SI（COMEX白银期货） | 白银现货 | 极高 | 同上，波动大margin易上调 | 高β波段（仓位减半） |
| 沪金AU（上期所） | 国际金×USD/CNY+国内溢价 | 高 | 人民币升值→沪金相对偏弱 | 国内时段主力，内外盘套利 |
| 沪银AG | 同上+工业属性 | 极高 | 同上 | 高β，仓位更小 |
| GLD/IAU（美股黄金ETF） | 黄金现货减费用 | 无（除非margin） | 管理费0.25-0.4%，tracking error | 长线配置/期权底层 |
| SLV | 白银现货 | 无 | 管理费+托管风险 | 银β暴露 |
| 纸黄金（银行） | 银行挂牌价 | 低/无 | 买卖价差大，无做空 | 无杠杆长线囤金 |
| 黄金期权（GC/GLD/沪金期权） | 非线性：方向+波动率+时间 | 可控 | theta（时间损耗）最大敌人 | 替代期货减回撤/事件押注/收入策略 |

### 4.2 内外盘价差规则

```
沪金 ≈ 国际金(USD/oz) × USD/CNY ÷ 31.1035 + 国内溢价
```

**操作含义**：
- 人民币预期贬值 → 沪金溢价走高 → 可卖沪金/买国际金（对冲）
- 国内进口配额收紧 → 溢价飙升 → 方向跟随但节奏不同
- 每日必须核对一次隐含汇率溢价，避免误判

### 4.3 载体选择决策树

```
宏观方向确定后，依次回答：

1. 需要杠杆表达还是低摩擦持有？
   → 杠杆：期货/现货margin；低摩擦：GLD/纸金

2. 有信心扛回撤吗？
   → 有信心+趋势清晰：期货单边；没信心：期权买Call/Put或期货+保护Put

3. 事件窗（CPI/FOMC/地缘）？
   → 是：避开高杠杆过夜，或轻仓，或期权结构

4. 做套利吗？
   → 金银比：两边都用期货腿（速度一致）；内外盘：两边都需流动性
```

## 第五章 多周期执行规则

### 5.1 周期角色定义

| 周期 | 职责 | 错误用法 |
|------|------|----------|
| **日线** | Regime判断、关键S/R（200ma）、CFTC/ETF背景 | 用来找抄底点（太粗） |
| **4H** | 界定当前结构（趋势/箱体）、画供需区、定止损结构位 | 频繁进出（磨损大） |
| **1H** | 回踩确认（pin/吞没/成交量柱）、挂单区 | 与4H冲突时硬做 |
| **15m** | 精细化入场、分批挂单、最后一档止损微调 | 用15m指标"发现趋势" |

### 5.2 层级纪律（写死）

**规则A：15m不能开新逆Regime仓位**
即使15m出现完美RSI超卖+pin bar，若Level 1是R3（鹰派压制）且价格在200日下方→只能做存量空单的获利减仓，不允许开新多。

**规则B：4H不确认，1H/15m信号降级为"管理单"而非"建仓单"**
4H仍在均线缠绕/结构内→1H信号最多轻仓1/3，且当天必须平或移至盈亏平衡止损。

**规则C：同方向多周期共振才给"满配仓位"**
```
满配仓位 = 日线方向✅ AND 4H结构到位✅ AND (1H触发✅ OR 回踩到价✅)
缺任何一个→仓位打折扣（×0.5或×0.33）。
```

## 第六章 期权专项应用

期权不是"赌波动"，而是方向表达的另一种载体。根据Regime选择期权结构。

### 6.1 四种基本用途

| 用途 | 结构 | 适用场景 | 注意事项 |
|------|------|----------|----------|
| **替代期货（有限风险）** | 买Call / 买Put | 方向有信心，但怕黑天鹅/数据滑点 | 选流动性好的月份（如GC近月或次近月） |
| **保护性保险** | 期货多单 + 买Put | 趋势多但事件窗临近 | 成本=期权Premium，需评估性价比 |
| **收入/卖出时间** | Covered Call / Cash-Secured Put / Iron Condor | R2横盘市，隐含Vol偏高 | 高阶策略，需懂Greeks管理，设置硬止损 |
| **事件押注（双买）** | Long Strangle / Straddle | 事件前方向不明但确信要大动 | 事件后尽快平仓（IV Crush） |

### 6.2 Regime对应期权选型

| Regime | 推荐期权策略 | 禁用策略 |
|--------|-------------|----------|
| **R1** | 买Call（替代期货）或期货+保护Put | 卖Naked Call |
| **R2** | Covered Call（持有GLD）、Iron Condor（高手）、卖Credit Spread | 裸买Gamma（Theta吃死） |
| **R3** | 买Put或反弹后Covered Call against GLD持仓 | 裸买Call |
| **R4** | 买远端Call（时间拉长，Theta压力小），仓位轻 | 卖Vol（方向不明） |
| **R5** | 什么都不做 | 所有期权（流动性枯竭） |

### 6.3 期权仓位风控

**买权（Long Call/Put）**：
- 单笔期权Premium ≤ 账户 × 0.5%–1.5%
- 总期权敞口（未平仓Premium总值） ≤ 账户 × 5%

**卖权（Short Vol）**：
- 必须用SPAN Margin计算
- 卖Vol账户至少预留40%-50%现金缓冲

### 6.4 波动率指数在期权策略中的应用

**GVX指导黄金期权策略**：
- GVX < 13.5（<25%分位）：买权极便宜，适合买Call/Put，成本最低
- GVX 13.5-18.2（25%-50%分位）：买权便宜，适合买Call/Put
- GVX 18.2-24.8（50%-75%分位）：正常水平，标准策略
- GVX 24.8-31.5（75%-90%分位）：买权贵，卖权收入高，适合卖Covered Call/Iron Condor
- GVX > 31.5（>90%分位）：极高波动，暂停期权交易，等待回归
- GVX > 38.2（>95%分位）：极端波动，流动性危机，禁止所有期权交易

**VXSLV指导白银期权策略**：
- VXSLV < 18.5（<25%分位）：买权极便宜，适合买Call/Put
- VXSLV 18.5-26.3（25%-50%分位）：买权便宜，适合买Call/Put
- VXSLV 26.3-35.8（50%-75%分位）：正常水平，标准策略
- VXSLV 35.8-45.2（75%-90%分位）：买权贵，卖权收入高
- VXSLV > 45.2（>90%分位）：极高波动，暂停期权交易
- VXSLV > 52.5（>95%分位）：极端波动，流动性危机，禁止所有期权交易

**GVX/VXSLV价差信号**：
- GVX > VXSLV：黄金不确定性高于白银，关注黄金方向
- VXSLV > GVX × 1.5：白银波动率溢价过高，可能回归
- GVX/VXSLV同步上升：市场整体不确定性上升，减仓
- GVX/VXSLV同步下降：市场恢复平静，可逐步加仓

**GVX/VXSLV价差信号**：
- GVX > VXSLV：黄金不确定性高于白银，关注黄金方向
- VXSLV > GVX × 1.5：白银波动率溢价过高，可能回归

## 第七章 风险管理体系

### 7.1 仓位分级（硬性上限）

| 品种 | 单边趋势 | 震荡行情 | 事件窗口 |
|------|----------|----------|----------|
| 黄金（期货/现货） | 总资金5%-10% | 减半（2.5%-5%） | 再减半（1.25%-2.5%） |
| 白银（期货/现货） | 总资金2%-5% | 1%-2.5% | 0.5%-1.25% |
| 期权买权Premium | 0.5%-1.5%/笔 | 同上 | 同上 |
| 期权卖权保证金 | 按SPAN，预留40%现金 | — | — |

### 7.2 止损硬性规则

- **趋势单**：止损放在关键高低点外侧（结构位），不轻易移动。
- **短线日内**：止损不超过单笔资金2%（按账户权益算）。
- **期权买权**：最大亏损即为Premium，无需额外止损；但可设定时间止损（如持仓3天后未达预期则平仓）。

### 7.3 避开的高风险窗口

- 美联储议息、CPI、非农数据前1-2天：降低仓位，不重仓过夜。
- 数据行情滑点大，容易击穿止损，建议使用限价单或期权保护。

### 7.4 流动性监控（R5检测）

每日检查以下指标，任何一项亮红灯则进入R5警戒，只平仓不开新仓：

```
□ SOFR利率是否跳升超过5bp/日
□ 美元交叉货币基差（EUR/USD、JPY/USD）是否急剧走阔
□ 黄金租赁利率（GOFO）是否异常升高
□ COMEX注册库存（Eligible→Registered）是否骤降
□ GVX是否急升突破31.5（>90%分位）或38.2（>95%分位）
□ VXSLV是否急升突破45.2（>90%分位）或52.5（>95%分位）
```

## 第九章 常见误区与修正（系统内置纠偏）

| 误区 | 系统如何防止 |
|------|-------------|
| 单纯靠避险无脑做多 | Regime诊断中R5除外，避险脉冲必须结合流动性检查，且禁止长线持仓 |
| 同等仓位做白银 | 白银仓位上限强制为黄金的一半 |
| 加息周期长期抄底黄金 | R3下央行抄底策略休眠，宏观利率波段只做空 |

## 第十章 仓位计算与风险管理

### 10.1 仓位计算公式

每次开仓前必须计算，不允许"拍脑袋"决定手数：

> **手数 = (账户权益 × 单笔风险%) ÷ (止损点数 × 每点价值)**

**参数说明**：
- **单笔风险%**：建议1%~2%（激进者不超过2%，保守者1%）
- **止损点数**：入场价−止损价（绝对值）
- **每点价值**：因品种和合约而异

**各品种每点价值参考**：

| 品种 | 合约规格 | 每点价值 | 最小变动 | 备注 |
|------|---------|---------|---------|------|
| 现货XAUUSD | 1手=100盎司 | $100/点 | $0.01 | 杠杆平台，注意swap |
| COMEX GC | 1手=100盎司 | $100/点 | $0.10 | 保证金约$10,000~14,000/手 |
| 沪金AU | 1手=1000克 | 1000元/点（¥1/克） | ¥0.02 | 保证金约12~18% |
| 沪银AG | 1手=15千克 | 15元/点（¥1/千克） | ¥1 | 保证金约12~18% |
| TD Au(T+D) | 1手=1000克 | 1000元/点 | ¥0.01 | 注意递延费 |

**计算示例**：
- 账户30万，单笔风险1.5%，止损8元/克（沪金）
- 手数 = 300,000 × 1.5% ÷ (8 × 1000) = 4,500 ÷ 8,000 = **0.56手 → 取0.5手**

### 10.2 风控铁律

| 规则 | 标准 | 原因 |
|------|------|------|
| 单笔最大亏损 | 账户权益的1%~2% | 保证连续止损10次仍存活 |
| 单日最大亏损 | 账户权益的3%~5% | 防止情绪化连亏 |
| 周度最大亏损 | 账户权益的5%~8% | 强制暂停，冷静复盘 |
| 连续亏损3次 | 强制减仓50%或暂停 | 说明市况与策略不匹配 |
| 单品种最大暴露 | 总权益的20% | 分散风险 |
| 总杠杆仓暴露 | 总权益的40%~60% | 防系统性风险 |

### 10.3 止损设定规则

**止损 = 结构破坏位 + 缓冲**，不是"随便设个数字"：

| 止损类型 | 设定方法 | 适用场景 |
|----------|---------|---------|
| **结构止损** | 前低/前高 ± 0.5~1×ATR(14) | 趋势回踩、突破 |
| **ATR止损** | 入场价 ± 1.5×ATR(14) | 波动率止损，通用 |
| **均线止损** | 收盘跌破MA20(4H)或MA50(日线) | 趋势跟踪 |
| **时间止损** | 入场后N根K线未达预期→平仓 | 防止"死扛等方向" |

**铁律**：止损价在入场前就定好，写下来，成交后绝不移动止损"给它多一点空间"。

### 10.4 出场管理规则

**分批止盈 + 移动止损**（不允许"一把梭进去、一把梭出来"）：

| 阶段 | 动作 | 条件 |
|------|------|------|
| **第一目标** | 减仓30%~50% | 到达前高/前低/Fib 38.2%回撤/1倍ATR利润 |
| **保本止损** | 剩余仓止损提到入场价+1跳 | 第一目标达成后 |
| **第二目标** | 再减仓30% | 到达Fib 61.8%/2倍ATR利润/关键结构位 |
| **跟踪止损** | 剩余仓用MA20(4H)或前一波低点跟踪 | 让利润奔跑 |

## 第十一章 策略型交易建议输出规范

操作建议必须输出**策略型交易建议**——不是模糊的"偏空观察"，而是可直接执行的策略场景，含完整参数。

### 11.1 核心原则

1. **策略场景制**：每个品种输出2~3个按优先级排列的策略场景（Plan A/B/C），而非单一方向判断
2. **参数完整**：每个场景必须包含入场价、止损价、目标T1/T2、R:R比、仓位手数
3. **条件触发**：用"若…则…"句式描述触发条件，但条件之后必须给出**具体数字**
4. **作废条件**：每个场景必须有"若反向突破XX则该场景作废"
5. **优先级排序**：最可能发生的场景排第一，备选场景排第二/第三

### 11.2 策略场景标准格式

```

  ├─ 触发条件：若价格[具体动作]至____附近 + [确认信号]（来源：____）
  ├─ 入场价：____
  ├─ 止损价：____（距入场____点，= ____×ATR(14)）
  ├─ 目标T1：____（到T1减仓____%，R:R = 1:____）
  ├─ 目标T2：____（剩余仓跟踪止损至____）
```

### 11.4 操作建议铁律

1. **必须给出具体数字**——"偏空观察，目标3300"是废话；"入场3305，止损3330，目标3270"才是交易建议
2. **必须按优先级排序**——策略A是最可能执行的，B是备选，C是对冲/极端场景
3. **必须用"若…则…"触发**——但触发之后必须跟具体参数，不是"偏空观察"
4. **必须有作废条件**——每个策略场景都有"认错线"
5. **必须有仓位计算**——不能只说"轻仓"，要说"0.4手"
6. **必须有R:R**——每个场景的风险收益比必须≥1:1.5（趋势强时≥1:2）
7. **必须标注价位来源**——"S1=3340（前低+MA20(4H)重合）"
8. **必须标注Regime和策略编号**——明确该策略属于哪种Regime、哪种策略类型
9. **禁止"偏多观察/偏空观察"作为最终建议**——观察不是建议，策略才是

## 第十二章 策略组合优化

### 12.1 策略组合优化框架

**目标**：研究多策略组合的风险收益特征，分析策略间的相关性，优化策略组合的权重。

**核心方法**：
- 均值-方差优化（Markowitz模型）
- 夏普比率最大化
- 分散化收益分析

### 12.2 策略相关性分析

**相关性矩阵**：
- 计算各策略间的相关性系数
- 识别低相关性或负相关性的策略对
- 优化策略组合的分散化效果

**分散化收益**：
- 平均相关性越低，分散化效果越好
- 分散化比率 = 1 / (1 + 平均相关性 × (策略数-1))

### 12.3 策略组合权重优化

**等权重基准**：
- 所有策略等权重配置
- 作为基准比较

**优化权重**：
- 基于夏普比率最大化
- 考虑策略的风险收益特征
- 动态调整权重

### 12.4 策略组合分析

**Regime适配**：
- 根据当前Regime选择适用策略
- 按权重排序，优先执行高权重策略

**组合收益**：
- 计算组合的预期收益和风险
- 评估组合的风险调整后收益

## 第二十一章 风险监控增强

### 21.1 风险监控框架

**目标**：实时风险计算和预警。

**核心功能**：
- VaR计算
- CVaR计算
- 最大回撤计算
- 风险指标计算
- 风险预警

### 21.2 VaR计算

**VaR（风险价值）**：
- 定义：在给定置信水平下，资产在特定时间内的最大可能损失
- 计算方法：历史模拟法
- 参数：置信度95%，时间窗口1天

**计算公式**：
```
VaR = |排序后收益的(1-置信度)分位数| × √时间窗口
```

### 21.3 CVaR计算

**CVaR（条件风险价值）**：
- 定义：在给定置信水平下，资产在特定时间内的平均损失
- 计算方法：历史模拟法
- 参数：置信度95%

**计算公式**：
```
CVaR = |排序后收益的前(1-置信度)个的平均值|
```

### 21.4 最大回撤计算

**最大回撤**：
- 定义：从峰值到谷值的最大跌幅
- 计算方法：历史模拟法

**计算公式**：
```
最大回撤 = (峰值 - 谷值) / 峰值
```

### 21.5 风险指标计算

**风险指标**：
- 夏普比率
- 索提诺比率
- 卡尔玛比率

**计算公式**：
- 夏普比率 = (年化收益 - 无风险利率) / 年化波动率
- 索提诺比率 = (年化收益 - 无风险利率) / 下行波动率
- 卡尔玛比率 = 年化收益 / 最大回撤

**预警级别**：
