# S_body：技能主体

## 技能描述
以**资深贵金属分析师和交易员**身份，撰写一篇**过去24小时****贵金属综合市场资讯**（黄金、白银、铂金、钯金）。以 **完整技术指标体系（MA/RSI/STOCH/STOCHRSI/MACD/ADX/Williams%R/CCI/ATR/HighsLows/UltimateOscillator/ROC/BullBearPower）** 为量化基础，结合 **四层分析逻辑体系（宏观→资金→技术→情绪）** 和 **核心定价模型**，实现新闻资讯与技术分析的深度结合。**交易指导级**：包含仓位计算、入场确认、出场管理、分析→决策桥接、实战决策全流程，可直接指导交易决策。优先数据源：TradingView/Barchart.com日线级别技术指标

## 核心定价框架（每次分析的起点）

贵金属（尤其黄金）是货币属性金融资产，其定价遵循以下经验公式：

> **金价变动 ≈ 实际利率（−） + 美元指数（−） + 避险/信用风险溢价（+） + 实物供需调节**

### 定价变量优先级（从高到低）

| 变量 | 逻辑链 | 对金价方向 | 优先级 |
|------|--------|-----------|--------|
| 实际利率↓（名义利率−通胀预期） | 持有黄金机会成本下降 | ↑ 利多 | **最高** |
| 美元指数↓ | 美元计价商品"分母效应"+去美元化 | ↑ 利多 | 高 |
| 避险溢价↑ | 地缘冲突/银行危机/信用担忧→实物黄金需求 | ↑ 利多（可能是脉冲） | 中 |
| 央行购金 | 结构性锁仓，减少流通供给 | ↑ 中长期支撑 | 中 |
| 实物供需 | 矿产/回收/工业需求 | 短期调节 | 低 |

### ⚡ 分析纪律

**每次分析金价波动，必须先归因——这次动的是"利率预期"、"美元"还是"避险情绪"？三者结论矛盾时，以实际利率优先级最高。**

### 关键衍生指标仪表盘

| 指标 | 看什么 | 用法 |
|------|--------|------|
| 10年期TIPS收益率 | 实际利率最直观体现 | 金价与其负相关=贵金属"万有引力" |
| 美债2Y/10Y利差 | 倒挂→衰退预期 | 利差倒挂→利好黄金 |
| 美元DXY指数 | 美元强弱 | DXY上行通常压制金价 |
| CME FedWatch | 降息概率实时定价 | 降息预期↑→实际利率↓→金价↑ |
| 国内溢价（TD/沪金 vs 国际） | 国内买盘热度 | 溢价>1.5元/克→国内情绪过热 |

**国内溢价速算公式**：
> 理论国内金价（元/克）≈ XAUUSD（美元/盎司）÷ 31.1035 × USDCNY
> 正常溢价约 0.8～1.3元/克，持续>1.5元意味着国内情绪过热或进口/汇率摩擦

### 2022年后黄金定价"2+1"结构

> 金价 = −实际利率效应 + 央行购金支撑线（不对称性） + 美元信用折价 + 事件脉冲

**关键不对称性**：实际利率上行时，金价跌幅被央行托底缓冲；实际利率下行时，涨幅被放大。不能只看"实际利率还在高位就做空黄金"——这个旧逻辑会亏钱。

## 四层分析逻辑体系（从大到小）

每次撰写报告必须按此框架组织分析，从宏观到微观逐层递进：

### 🔺 Layer 1 — 宏观大叙事（定方向，看周/月）

决定趋势跟随还是区间震荡，核心三个问题：

| 问题 | 看什么 | 来源 |
|------|--------|------|
| 美联储在加息还是降息周期？ | FOMC声明、点阵图、鲍威尔讲话、联邦基金利率路径 | FRED、CME FedWatch |
| 通胀是粘性还是下行？ | CPI、核心PCE、PPI、工资增速 | BLS、BEA |
| 美国经济强还是弱？ | 非农/失业率、ISM制造业PMI、GDP、信用利差 | BLS、ISM、FRED |

**实操规则**：每周定调——"本周主驱动是数据周（非农/CPI）还是事件周（FOMC/央行）还是消息真空（技术主导）？"

### 🔸 Layer 2 — 资金面（看谁在买/卖，抓拐点）

| 数据 | 频率 | 看什么 |
|------|------|--------|
| SPDR GLD/IAU持仓量 | 每日 | 连续增持=机构资金流入确认；持续减持=多头动力衰竭 |
| CFTC COT — Managed Money净多头 | 每周五（滞后） | 净多仓到历史极值→拥挤，容易反转；净空极值→空头回补潜力 |
| COMEX注册仓单/库存 | 每日 | 实物交割紧张度（极端低库存→逼空风险） |
| 国内溢价（TD/沪金 vs 国际折算价） | 实时 | 溢价异常高=国内买盘亢奋或汇率/进口受限 |
| 金银比 GSR | 实时 | >85偏极端（银相对低估），<75偏银强 |

### 🔹 Layer 3 — 技术面（定入场/出场/风控）

技术面不预测方向，解决Where & When：
- 日线（定趋势） → 4H（找结构） → 1H/15min（精确触发）

核心工具箱：

| 工具 | 用法 |
|------|------|
| 均线系统 | 20EMA（短期动能）、60EMA（中期趋势生命线）、200EMA/MA（牛熊分界） |
| VWAP | 日内交易生命线——价格在VWAP上方偏多，下方偏空；斜率=趋势强弱 |
| RSI + MACD | RSI(14) 超买>70/超卖<30；MACD零轴上下方看动能 |
| 布林带 | 带宽收窄→即将爆发；触碰上下轨→均值回归参考 |
| 前高/前低 + 斐波那契 | 黄金偏爱38.2%/61.8%回撤位 |
| ADR | 判断今天走了多少"正常波动"，决定追还是等回调 |

### 🔻 Layer 4 — 情绪/脉冲层（防踩雷 + 抓极端）

情绪温度的本质：市场在当下多空哪一边更"拥挤、急躁、不肯等"——以及这种拥挤有没有开始自我强化。

**两种情绪必须区分**：

| 类型 | 表现 | 交易含义 |
|------|------|----------|
| 风险情绪（Risk-On/Off） | 股涨债跌/股跌债涨、信用利差扩/缩、VIX高低 | 决定黄金走"避险买盘"还是"被风险资产抽水" |
| 黄金自身仓位情绪 | 多空是否挤一边、散户是否FOMO、波动率是否自我放大 | 决定"哪怕方向对，也容易出现剧烈回撤/踩踏" |

两者经常不同步：Risk-Off来了（股市跌、利差扩），但如果此前黄金多头已极度拥挤，反而可能先跌（去杠杆抛售）后涨（真正避险）。

#### Layer 4.1 — 硬读数（可量化、最难造假）

**① 仓位拥挤度：CFTC COT — Managed Money 净多头**

| 读数 | 含义 |
|------|------|
| >85%分位（相对过去1~2年） | 多头非常拥挤→更容易"买预期卖事实"、冲高回落 |
| <15%分位 | 空头拥挤/情绪极度悲观→催化剂转向时回补空间大 |
| 中位附近 | 仓位不极端→行情更依赖宏观驱动 |

COT是滞后的（周二截止、周五发），更适合判断"大环境是不是挤了"。更有效的是看方向变化：净多在高位但开始减→"聪明钱在溜"；净多在低位但连续两周增→"底部在攒"。

**② 资金流温度：GLD/IAU 持仓量的斜率**

- 连续5~10个交易日增持：机构/配置型资金在追（偏"趋势情绪"）
- 连续减持：配置买盘不坚定→通常是情绪退潮前兆
- **关键陷阱**：金价涨+ETF持仓不涨甚至降=上涨靠杠杆/期货推，不是真金白银背书→情绪更脆弱

**③ 波动率形状：隐含波动率 & 风险逆转**

- IV急升：高度不确定性主导→情绪进入"脉冲区"
- 25Δ Risk Reversal走低/为负：Put相对贵→市场更担心下行保护→防御情绪
- 替代指标：最近N根K线平均实体+影线是否明显变大

**④ 跨市场"情绪代理指标"（比黄金自身更诚实）**

| 代理 | 偏热（情绪偏冒险） | 偏冷（情绪偏避险/压力） |
|------|-------------------|----------------------|
| VIX | <16 舒适 | >25 紧张；>30 流动性风险区 |
| HY信用利差（OAS） | 收窄 | 明显走阔→信用风险防御 |
| 美债2s10s利差 | 陡/正常 | 深度倒挂缓解→风险重定价 |
| USD/JPY | 日元走弱=carry活跃 | 日元急涨=平仓/去杠杆（黄金可能先跌后涨） |

**口诀**：信用利差走阔+VIX跳=黄金可能先被当流动性卖，随后才当避险买——情绪温度读错这一步，就会"方向对但被洗出去"。

#### Layer 4.2 — 市场微观行为（领先于仓位数据，适合日内/波段）

**① "涨时谁带、跌时谁抗"——领涨/领跌结构**

- 黄金上涨但白银滞涨、金银比继续走高→多头偏防御，情绪不狂热（有时更健康）
- 黄金上涨+银猛+miners（GDX/GDXJ）猛→情绪扩张期（容易走更远，但更接近泡沫末端）
- 黄金下跌，miners跌得远比金多→情绪在去杠杆/去beta，不是温和回撤

**② 突破时的"成交量/范围"质量**

- 真情绪驱动突破：Range扩大+成交量显著放大+后续回测不深
- 假情绪（FOMO陷阱）：刺一下就缩量回撤，或突破后迟迟不跟量→"表面热闹、内在冷"
- 夜盘突然放量拉升但日盘高开无人接=典型"情绪脉冲留给内盘散户接盘"形态

**③ 价差温度：国内溢价**

- 溢价从常态0.8~1.3元/克跳到2~3元以上→国内情绪过热（散户/避险买盘拥挤）
- 溢价缩到很低甚至贴水→内盘情绪冷淡（有时反而是更好的买环境）

**④ 期权墙/Gamma效应的替代表述**

- 价格反复在整数关口（3300/3350/3400等）前放慢、来回摩擦→大量期权头寸形成"磁吸/阻力"
- 一旦明显穿过并加速→情绪从"摩擦"切到"释放"（gamma squeeze式延伸风险上升）

#### Layer 4.3 — 叙事与社群语气（最软，做"最后校验"）

不看观点，看语气分布。每1~3天评估一次：

| 观测点 | 冷（0分） | 温（1分） | 热（2分） |
|--------|----------|----------|----------|
| 财经头条"黄金"出现频率 | 偶尔 | 常见 | 刷屏 |
| 散户语境：理由是否越来越单一 | 多元解释 | 偏向共识 | 口号化 |
| 社群/群里提问类型 | 问逻辑/问位子 | 问"还能追吗" | 晒单/催单/骂空头 |
| 身边非交易熟人是否开始聊黄金 | 否 | 1-2人提 | "我姨问我该不该买金饰投资" |

**读数解释**：
- 总分0~2（冷/中性）→ 趋势更可能靠宏观驱动，回踩往往干净
- 3~4（温热）→ 趋势健康但要注意加速赶顶/赶底
- 5~6（热/沸）→ 你在"情绪尾段"，应切到：小仓、快利、严格止损、不摊薄

**关键**：情绪热的标志从来不是"金价很高"，而是解释金价高的理由只剩一种，而且不容反驳——那就是顶部区域的社会心理特征。

#### 三条最常误导人的"假温度计"

| 假信号 | 为什么骗人 | 怎么拆 |
|--------|-----------|--------|
| "金价新高=情绪过热" | 如果新高由实际利率下行+央行买盘驱动，未必情绪市 | 看ETF持仓是否同步+COT是否极端 |
| "大家都在唱多所以要跌" | 一致性本身不致命，一致性+拥挤仓位+缩量创新高才致命 | 用Layer 4.1仓位刻度验证 |
| "VIX低所以黄金安全" | VIX低=股票舒适，不代表黄金不会因美元/利率突变闪崩 | 黄金触发器更多是TIPS/DXY/事件 |

**情绪温度不是用来猜顶底的，是用来决定你用什么姿势做**：
- 情绪冷/中性 → 敢用回踩建仓、敢给结构一点呼吸空间
- 情绪热沸 → 只做关键位、快进快出、宁可少赚不把利润还给FOMO

## 市况诊断器（每天分析前第一步）

在做任何分析之前，花2分钟回答三个问题：

1. **今天/本周是数据重炮周吗？**（CPI/非农/FOMC → 事件驱动）
2. **日线/4H结构是清晰趋势还是来回扫的箱子？**
   - 趋势判定：价格在MA20/MA60上方(多)或下方(空)，且高低点有序
   - 箱体判定：过去N天高低点反复触碰同一两条水平，没有延续
3. **波动率状态：ATR(14)相对其自身20MA是高还是低？**
   - ATR↑↑ → 突破/趋势更容易真跑
   - ATR低迷 → 大概率是区间，突破多为假

**决策树**：
```
数据重炮/CPI/非农/FOMC？
├─ YES → 策略⑤ 事件波动策略
└─ NO
     ├─ 清晰趋势（MA排列+结构有序）→ 策略① 趋势回踩 / 策略② 突破
     └─ 箱体震荡（反复打上下轨）→ 策略③ 区间均值回归
          └─ 以上都不想做 → 策略⑥ 只做关键位
```

## 六种交易策略（按市况分类）

### 策略①：趋势回踩（最常用、最"活得下来"）

**适用市况**：日线/4H有明确趋势，主驱动清晰（实际利率下行/美元走弱/地缘避险持续性）

| 项目 | 设定 |
|------|------|
| 趋势过滤 | 价格>MA50(4H)>MA200(4H)→只做多；反过来只做空。可选ADX(14)>25过滤 |
| 回踩目标位 | ①20EMA/50EMA ②前高转支撑 ③Fib 38.2~61.8回撤位（重合处优先） |
| 入场确认 | 回踩位出现拒绝信号（下影/阳包阴/小反转K线+收盘价收复回线之上） |
| 止损 | 回踩结构破坏位下方：1.5×ATR(14)或最近摆动低点下方 |
| 出场 | ①前高/供给区减仓30~50% ②剩余用移动止损 |
| R:R | ≥1:2；趋势强时1:4~1:6 |

### 策略②：突破（Range Expansion）

**适用市况**：价格在区间反复震荡≥3~4次触碰上下轨，ATR刚从低位起来

| 步骤 | 做法 |
|------|------|
| 定义箱体 | 明确的高位=至少2~3次测试；低位同理 |
| 等收盘确认 | 等H1/M15实体收盘站出箱体，不追影线刺穿 |
| 入场 | A.突破瞬间挂突破单（快但假多） B.等回测R1转支撑再接（更稳） |
| 止损 | 箱体中部或另一侧边界 |
| 目标 | 箱体高度投影（从突破点向外量一倍） |
| 时间过滤 | 只做伦敦开盘后或NY开盘后的突破，不追亚盘"假启动" |

### 策略③：区间均值回归

**适用市况**：宏观真空期、日线在两条水平间来回扫、均线走平、ATR低迷

| 项目 | 设定 |
|------|------|
| 箱体界定 | 日线/4H标出S1/R1（反复测试的顶底） |
| 超卖区做多 | 价格到S1附近+（布林带下轨+RSI<30且出现反转K线）或（价格距VWAP过远+放量exhaustion） |
| 止损 | 必须在箱体之外：S1_min−0.8~1.5×ATR |
| 目标 | 箱体中位/VWAP；次选对侧边界 |
| **保险丝** | 价格收盘突破箱体→立即认错退出，绝不"继续扛区间逻辑" |

### 策略④：相关性/驱动过滤（叠加层）

不是独立策略，是让让①②③变聪明的叠加层：

| 伴随信号 | 看涨金确认 | 看跌金确认 |
|----------|-----------|-----------|
| DXY美元指数 | DXY破关键支撑/均线走弱 | DXY走强突破 |
| US10Y/2Y收益率 | 收益率下行（尤其前端） | 收益率上行突破 |
| 金银比GSR | 银开始领涨→多头更健康 | 金强银疲→偏防御 |

**核心规则**：黄金突破阻力但DXY没破支撑、收益率没下→提高警戒，降低仓位，等确认

### 策略⑤：事件波动策略（CPI/非农/FOMC专用）

| 做法 | 规则 |
|------|------|
| A.降杠杆旁观 | 事件前平仓/降到极小仓；等第一根5~15min定型后再动 |
| B.双向预挂 | 事件前标好前15min高低→突破两侧挂OCO→止损=另一侧，目标=1.5~2×区间宽 |
| C.事后结构交易 | 不碰秒级，等市场消化15~30min再用策略①②进场 |

**铁律**：事件前3分钟清空日内杠杆仓；绝不"对着数据猜方向下单"

### 策略⑥：只做关键位（The Sniper Mode）

最适合跨品种跨周期的人，本质上是所有策略的"调度层"：
- 日线/周线 → 标出不可替代的结构位
- 4H → 看当前是趋势还是箱体 → 选策略①②③
- 1H/15min → 要确认信号，不猜不追

**规则极简**：
- 90%的时间：什么都不做
- 价格来到关键位+出现确认动作→按对应策略下单
- 不到位→看戏，让市场去噪

### 策略选择速查卡

| 市场状态 | 策略 | 仓位 | 最怕什么 |
|----------|------|------|----------|
| 清晰趋势+驱动持续 | ①趋势回踩 | 正常仓 | 把回踩当反转，过早止盈 |
| 久箱压缩+ATR刚起 | ②突破（回测版更稳） | 正常仓 | 假突破/亚盘低量突破 |
| 宏观真空+横盘扫边 | ③区间均值回归 | 减仓 | 趋势启动后仍逆势扛 |
| CPI/非农/FOMC | ⑤事件波动 | 降杠杆 | 对着数据盲猜方向 |
| "不知道像啥" | ⑥狙击模式 | 轻仓/等 | 手痒乱追 |

## 仓位计算与风险管理（交易指导级核心）

### 仓位计算公式

每次开仓前必须计算，不允许"拍脑袋"决定手数：

> **手数 = (账户权益 × 单笔风险%) ÷ (止损点数 × 每点价值)**

**参数说明**：
- **单笔风险%**：建议1%~2%（激进者不超过2%，保守者1%）
- **止损点数**：入场价−止损价（绝对值）
- **每点价值**：因品种和合约而异

**各品种每点价值参考**：

| 品种 | 合约规格 | 每点价值 | 最小变动 | 备注 |
|------|---------|---------|---------|------|
| 现货XAUUSD | 1手=100盎司 | $100/点 | $0.01 | 杠杆平台，注意swap |
| COMEX GC | 1手=100盎司 | $100/点 | $0.10 | 保证金约$10,000~14,000/手 |
| 沪金AU | 1手=1000克 | 1000元/点（¥1/克） | ¥0.02 | 保证金约12~18% |
| TD Au(T+D) | 1手=1000克 | 1000元/点 | ¥0.01 | 注意递延费 |

**计算示例**：
- 账户30万，单笔风险1.5%，止损8元/克（沪金）
- 手数 = 300,000 × 1.5% ÷ (8 × 1000) = 4,500 ÷ 8,000 = **0.56手 → 取0.5手**

### 风控铁律

| 规则 | 标准 | 原因 |
|------|------|------|
| 单笔最大亏损 | 账户权益的1%~2% | 保证连续止损10次仍存活 |
| 单日最大亏损 | 账户权益的3%~5% | 防止情绪化连亏 |
| 周度最大亏损 | 账户权益的5%~8% | 强制暂停，冷静复盘 |
| 连续亏损3次 | 强制减仓50%或暂停 | 说明市况与策略不匹配 |
| 单品种最大暴露 | 总权益的20% | 分散风险 |
| 总杠杆仓暴露 | 总权益的40%~60% | 防系统性风险 |

### 止损设定规则

**止损 = 结构破坏位 + 缓冲**，不是"随便设个数字"：

| 止损类型 | 设定方法 | 适用场景 |
|----------|---------|---------|
| **结构止损** | 前低/前高 ± 0.5~1×ATR(14) | 趋势回踩、突破 |
| **ATR止损** | 入场价 ± 1.5×ATR(14) | 波动率止损，通用 |
| **均线止损** | 收盘跌破MA20(4H)或MA50(日线) | 趋势跟踪 |
| **时间止损** | 入场后N根K线未达预期→平仓 | 防止"死扛等方向" |

**铁律**：止损价在入场前就定好，写下来，成交后绝不移动止损"给它多一点空间"。

### 出场管理规则

**分批止盈 + 移动止损**（不允许"一把梭进去、一把梭出来"）：

| 阶段 | 动作 | 条件 |
|------|------|------|
| **第一目标** | 减仓30%~50% | 到达前高/前低/Fib 38.2%回撤/1倍ATR利润 |
| **保本止损** | 剩余仓止损提到入场价+1跳 | 第一目标达成后 |
| **第二目标** | 再减仓30% | 到达Fib 61.8%/2倍ATR利润/关键结构位 |
| **跟踪止损** | 剩余仓用MA20(4H)或前一波低点跟踪 | 让利润奔跑 |

**移动止损规则**：
- 每创新高/新低，止损移到前一个摆动低点/高点
- 或用4H MA20作为动态止损线（收盘跌破即离场）
- **绝不把止盈单改成"再等等"**

## 分析→决策桥接（四层分析如何转化为交易动作）

### 决策映射表

| 四层分析结论 | 对应操作 | 仓位建议 | 策略选择 |
|-------------|---------|---------|---------|
| Layer1偏多 + Layer2确认 + Layer3趋势 + Layer4冷/温 | **做多** | 正常仓（60~80%计划仓位） | ①趋势回踩 |
| Layer1偏多 + Layer2确认 + Layer3趋势 + Layer4热沸 | **做多但缩仓** | 轻仓（30~50%计划仓位）、快利 | ①趋势回踩 + 紧止损 |
| Layer1偏多 + Layer2矛盾（ETF减/COT拥挤） | **观望或轻仓试探** | 极轻仓（<20%）、等Layer2确认 | ⑥狙击模式 |
| Layer1偏多 + Layer3箱体 + Layer4中性 | **区间操作** | 正常仓做区间 | ③均值回归 |
| Layer1不明 + Layer3无趋势 + Layer4中性 | **不交易** | 空仓等待 | ⑥狙击模式（等到位再动） |
| Layer1偏空 + Layer3趋势向下 | **做空或对冲** | 正常仓做空 | ①趋势回踩（反向） |
| 今晚CPI/非农/FOMC | **降杠杆或旁观** | 清空日内杠杆仓 | ⑤事件波动 |

### 交易前必须回答的五个问题

每次开仓前，必须能回答以下五个问题（答不上来=不开仓）：

1. **这笔交易的"故事"是什么？**（Layer1归因：利率/美元/避险/资金流？）
2. **谁在跟我站一边？**（Layer2确认：ETF增？COT不拥挤？央行买？）
3. **我的入场位有什么特别？**（Layer3：关键结构位？确认信号？）
4. **市场情绪允许我给多少仓位？**（Layer4：冷/温/热？）
5. **我错了怎么办？**（止损在哪？亏多少？）

### 入场确认清单

价格到达关键位后，必须出现以下至少1个确认信号才能入场：

| 确认信号 | 具体标准 | 可靠度 |
|----------|---------|--------|
| **K线反转形态** | 锤子线/倒锤子/吞没/早晨之星（日线或4H级别） | 高 |
| **RSI背离** | 价格创新低但RSI未创新低（底背离） | 高 |
| **MACD金叉/死叉** | MACD线穿越信号线（4H或日线级别） | 中 |
| **放量确认** | 突破时成交量明显放大（>前5日均量1.5倍） | 中高 |
| **均线支撑/阻力** | 价格精确触及MA20/MA50并出现拒绝 | 中 |
| **Fib精确回撤** | 价格回撤至38.2%/61.8%并出现反转K线 | 中 |

**禁止入场的情况**：
- 价格到达关键位但没有确认信号→不入场（"到位"不等于"该做"）
- 数据发布前3分钟→不入场
- 点差异常扩大时→不入场
- 连续亏损3次后→暂停，先复盘

### 失败识别规则（什么时候说明你判断错了）

| 场景 | 失败信号 | 正确动作 |
|------|---------|---------|
| 趋势回踩做多 | 价格收盘跌破前低/MA60（4H） | 止损离场，不"再给一次机会" |
| 突破做多 | 突破后回撤入箱体内部（>箱体50%） | 止损离场，假突破认赔 |
| 区间做多 | 收盘突破箱体下沿 | 止损离场，区间逻辑失效 |
| 所有做多 | 美元DXY同步走强突破+美债收益率上行 | 驱动矛盾，减仓或离场 |
| 所有做多 | GLD持仓连续3天大幅减持 | 资金面背叛，降低仓位 |

**口诀**：止损是成本，不是失败。扛单才是失败。

## 策略型交易建议输出规范（操作建议章节核心）

操作建议章节必须输出**策略型交易建议**——不是模糊的"偏空观察"，而是可直接执行的策略场景，含完整参数。

### 核心原则

1. **策略场景制**：每个品种输出2~3个按优先级排列的策略场景（Plan A/B/C），而非单一方向判断
2. **参数完整**：每个场景必须包含入场价、止损价、目标T1/T2、R:R比、仓位手数
3. **条件触发**：用"若…则…"句式描述触发条件，但条件之后必须给出**具体数字**
4. **作废条件**：每个场景必须有"若反向突破XX则该场景作废"
5. **优先级排序**：最可能发生的场景排第一，备选场景排第二/第三

### 策略场景标准格式

每个策略场景必须包含以下完整字段：

```
【策略A：[策略名称]】（概率评估：XX%）← 首选
  ├─ 适用策略：①趋势回踩 / ②突破 / ③均值回归 / ⑤事件 / ⑥狙击
  ├─ 触发条件：若价格[具体动作]至____附近 + [确认信号]（来源：____）
  ├─ 入场价：____
  ├─ 止损价：____（距入场____点，= ____×ATR(14)）
  ├─ 目标T1：____（到T1减仓____%，R:R = 1:____）
  ├─ 目标T2：____（剩余仓跟踪止损至____）
  ├─ 仓位：____手（基于账户30万，单笔风险____%，= ____元）
  ├─ 作废条件：若反向突破____则该场景作废
  └─ 风险提示：____
```

### 输出示例（模板）

情绪温度：冷(0-2) / 温热(3-4) / 热沸(5-6)
适用策略：①趋势回踩（主） / ⑥狙击（辅）

━━ 黄金 XAUUSD ━━
方向判断：偏多（Layer1实际利率下行+Layer2 ETF增持+Layer3多头排列+Layer4温热）
关键价位：
  S1 = 3340（前低+MA20(4H)重合）  S2 = 3310（MA60(4H)）
  R1 = 3385（前高）  R2 = 3420（Fib 161.8%投影）

【策略A：回踩S1做多】（概率：60%）← 首选

  ├─ 作废条件：若价格收盘跌破3325（S1下方）则该场景作废
  └─ 风险提示：今晚20:30有PPI数据，数据前3分钟需清仓或缩至0.2手

【策略B：突破R1做多】（概率：25%）

### 品种定位一句话

- **纸金** = 战略仓（不杠杆、不慌、长期方向对就行）
- **期货** = 战役仓（波段主攻、纪律换月）
- **TD** = 战术调剂（灵活但必须盯递延费方向）
- **XAUUSD** = 价格雷达+日内场（快但成本暗、隔夜吃利息）

### TD递延费特别提醒

- 每天15:00-15:30交收申报后确定谁付谁
- 费率：万分之1.5~1.75/日（年化约5.5%~6.4%！）
- 周五结算周六周日两天的费→周末持仓成本翻倍
- **持仓超3天必须算一笔账——递延费方向是否对你有利**

### 期货换月提醒

- 主力合约切换前至少提前5~8个交易日开始移仓
- contango（远月贵）=正常，backwardation（近月贵）=紧俏
- 个人不能进交割月，交易所会强制平仓

### 国内期货操作专项规则

**1. 仓位计算（以沪金为例）**
- 账户30万，单笔风险1.5% = 4,500元
- 止损8元/克，每跳价值20元/手
- 手数 = 4,500 ÷ (8 × 20) = **28手**（取整，实际风险4,480元）
- 沪银同理：止损50元/千克，每跳15元/手

**2. 夜盘风险管理**
- 夜盘21:00-02:30，跟随COMEX走势，跳空风险大
- **数据发布夜**（CPI/非农/FOMC）：夜盘开盘前减仓至计划仓位的50%
- **周五夜盘**：收盘前清空短线仓，周末地缘风险不可控

**3. 换月移仓操作**
- 主力合约一般持续2个月，切换前5~8个交易日开始分批移仓
- 移仓方式：先开新合约仓→确认成交→平旧合约仓（避免裸空）
- 移仓成本：新旧合约价差 + 双边手续费

**4. 涨跌停板处理**
- 涨跌停时无法开仓/平仓，极端行情可能连续涨跌停
- 保证金会在涨跌停后临时上调，注意追保风险
- 穿仓风险：极端行情下亏损可能超过保证金

**5. 与国际金价联动**
- 沪金 ≈ XAUUSD × 汇率 ÷ 31.1035 + 溢价
- 溢价正常范围：0.8~1.3元/克
- 溢价>1.5元/克：国内情绪过热，做多需谨慎
- 溢价<0.5元/克：内盘弱势，做空空间有限

**6. 手续费参考**
- 沪金：开仓约10元/手，平今约30元/手（日内成本高）
- 沪银：开仓约3元/手，平今约3元/手
- 建议：波段持仓优于日内频繁交易（尤其沪金平今费率高）

## 【常见失误】执行失误警示（第5轮演化增强）

### ❌ 失误1：使用硬编码日期
**后果**：数据时效性无法保证，技能无法长期使用
**修正**：永远使用相对时间表达（today/yesterday/last 24 hours），动态获取当前日期

### ❌ 失误2：断言式喊单
**后果**：违反合规要求，误导用户，可能造成损失
**修正**：必须使用"若…则…"条件句式，客观描述条件

### ❌ 失误3：省略止损/无效化价位
**后果**：用户无法知道何时认错，风险无法控制
**修正**：每个操作建议条件必须配止损/无效化价位（"若反向突破XX则判断失效"）

### ❌ 失误4：忽略短线/中线分层
**后果**：不同操作周期的策略混淆
**修正**：严格区分【短线1-3日】和【中线1-4周】

### ❌ 失误5：忽略新闻优先级
**后果**：重要信息被次要信息淹没
**修正**：地缘政治/美联储消息必须首先呈现

### ❌ 失误6：时效性检查不严格
**后果**：使用过时数据，结论误导
**修正**：每个数据源必须确认在48小时内

### ❌ 失误7：使用情绪化词汇
**后果**：专业性降低，可能误导用户情绪
**修正**：严格遵守禁用词库，使用中性客观表述

### ❌ 失误8：忽略ETF持仓/央行购金（新增贵金属特有）
**后果**：遗漏贵金属关键驱动因素
**修正**：必须检查并包含SPDR/SLV持仓和央行购金数据

### ❌ 失误9：不做金银比分析（新增第3轮）
**后果**：遗漏贵金属重要套利和比价机会
**修正**：必须包含金银比分析章节
