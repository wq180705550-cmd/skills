### OVX指数分析框架
- **相对位置分析**（主策略）：
  - 近252日百分位：<20%=极低波动、20-80%=中性波动、80-95%=高波动、>95%=极值恐慌
  - OVX自身RSI(14)：>70=恐慌过度，<30=恐慌不足
  - 方向变化：单日>5点骤升=恐慌加剧，单日>5点骤降=恐慌缓解
- **绝对阈值分析**（辅助+风控红线）：
  - <20：绝对低波动
  - 20-30：历史中性区间
  - >35：一级风控线
  - >45：二级红线，系统性风险

## 四层分析框架

### 第一层：供需平衡表
| 维度 | 关键变量 | 看什么 |
|------|---------|--------|
| **供给端** | OPEC+产量政策 | 减产/增产配额、履约率 |
| | 美国页岩油 | 钻机数、完井数、DUC |
| | 地缘中断 | 中东冲突、利比亚内乱、俄罗斯制裁 |
| **需求端** | 全球GDP增长 | IMF/WTO经济展望 |
| | 季节性 | 夏季驾驶季、冬季取暖油 |
| | 替代与效率 | EV渗透率、燃油经济性 |
| **库存** | OECD商业库存 | 库存降则涨，累则跌 |

### 第二层：期限结构分析
- **Contango**（远月 > 近月）→ 现货过剩 → 价格承压
- **Backwardation**（近月 > 远月）→ 现货紧缺 → 价格有支撑

### 第三层：宏观与美元/利率环境
| 宏观因子 | 传导机制 |
|---------|---------|
| 美元指数 DXY | 美元走强 → 油价承压（负相关≈-0.7） |
| 实际利率 | 高利率 → 商品估值压缩 |
| 衰退预期 | PMI<50 → 需求预警 |
| 通胀 | 油价是CPI核心项 → Fed政策反噬 |

### 第四层：资金面与仓位结构
- CFTC持仓报告：对冲基金/投机资金的净多头水位
- 极端信号：净多头创多年新高 → 反向脆弱

## WTI特有的微观结构陷阱

### 库欣罐容阈值效应
- **库欣库存 → 逼近罐容下限**：空头害怕没地方交货 → 近月被逼空
- **库欣堆满**：储油溢价飙升 → 曲线深度Contango → 价格螺旋下行
- **关键阈值**：库欣库存 < 5年均值10%以下或利用率 > 85%

### 到期移仓（Roll）的机械性扰动
- Contango深：被动"低卖高买"→ ETF净值额外磨损
- Backwardation深：被动"高卖低买"→ 反而有正carry
