### Step 1.5：原油核心技术指标与波动率数据获取

**优先数据源**：直接访问 https://cn.investing.com/technical/commodities-technical-analysis 获取所有技术指标的完整分析

**统一获取流程**：
1. 首先访问cn.investing.com技术分析页面，获取所有指标的完整数据
2. 若页面访问失败，使用WebSearch获取各指标实时数据
3. 若仍无法获取，基于K线数据进行指标计算推导
4. 所有指标数据必须包含：当前值、变化方向、信号状态

**原油期货合约处理**：
- 关注近月主力合约（标准为1-3个月后到期合约）
- 合约换月期间需关注展期成本
- 获取数据时标注当前主力合约月份

**期限结构分析**：
- Contango结构（近月低于远月）：标准出现在供应过剩或存储成本高时，偏空信号
- Backwardation结构（近月高于远月）：标准出现在供应紧张或需求旺盛时，偏多信号
- 期限结构变化是原油独有的重要分析维度

### 技术指标体系（20个指标）

**指标1：均线系统 MA(5,10,20,60)**
- 信号解读：
  - 多头排列（MA5>MA10>MA20>MA60）：需进一步判断走势方向
  - 空头排列（MA5<MA10<MA20<MA60）：需进一步判断走势方向
  - ⚠️ **排列+走势综合判断**（v6.7.0新增）：
    - 多头排列 + 两条均线都在上升 → **真多头**，强势趋势，偏多
    - 多头排列 + 两条均线都在下降 → **伪多头**，实际空头趋势，偏空
    - 多头排列 + 一升一降 → **弱多头**，趋势减弱，观望
    - 空头排列 + 两条均线都在下降 → **真空头**，弱势趋势，偏空
    - 空头排列 + 两条均线都在上升 → **伪空头**，实际多头趋势，偏多
    - 空头排列 + 一升一降 → **弱空头**，趋势减弱，观望
  - 判断方法：比较当前MA值与5根K线前的MA值，计算斜率方向
  - 金叉（短期均线上穿长期均线）：买入信号
  - 死叉（短期均线下穿长期均线）：卖出信号

**指标2：RSI(14) — 相对强弱指数**
- 关键阈值：>70 超买区、<30 超卖区、30-70 中性
- 交叉信号：从下穿越30=买入信号，从上穿越70=卖出信号，背离信号（价格创新高但RSI未创新高或相反）

**指标3：STOCH(9,6) — 随机指标**
- 关键阈值：>80 超买区、<20 超卖区、20-80 中性
- 信号解读：K线从下穿越D线=买入信号，K线从上穿越D线=卖出信号

**指标4：STOCHRSI(14) — 随机RSI**
- 关键阈值：>80 超买区、<20 超卖区、20-80 中性
- 信号解读：极端值提示超买超卖

**指标5：MACD(12,26,9) — 指数平滑异同移动平均线**
- 信号解读：
  - 金叉（MACD线从下穿过信号线）：趋势确认，偏多
  - 死叉（MACD线从上穿过信号线）：趋势确认，偏空
  - 背离信号（价格创新高但MACD未创新高或相反）

**指标6：ADX(14) — 平均趋向指标**
- 关键阈值：>25 强趋势、<20 无趋势、20-25 弱趋势
- 信号解读：+DI>-DI 多头方向，-DI>+DI 空头方向

**指标7：Williams %R(14) — 威廉指标**
- 关键阈值：>-20 超买区、<-80 超卖区、-80~-20 中性
- 信号解读：超买后卖出信号，超卖后买入信号

**指标8：CCI(14) — 商品通道指数**
- 关键阈值：>+100 超买、<-100 超卖、-100~+100 中性
- 交叉信号：穿越+100向下=做空信号，穿越-100向上=做多信号，穿越0轴=动能转换

**指标9：ATR(14) — 平均真实波幅**
- 信号解读：波动率大小，用于设置止损幅度

**指标10：Highs/Lows(14) — 高低点**
- 信号解读：价格区间参考，支撑阻力位

**指标11：Ultimate Oscillator(7,14,28) — 终极振荡指标**
- 关键阈值：>70 超买区、<30 超卖区、30-70 中性
- 信号解读：多周期动量综合判断

**指标12：ROC(12) — 变动率指标**
- 关键阈值：>5% 强势、<-5% 弱势、-5%~5% 中性
- 信号解读：动量变化率

**指标13：Bull/Bear Power(13) — 多空力量指标**
- 信号解读：多头强势或空头强势判断

**指标14：WTI期权隐含波动率(IV) — OVX指数**
- **相对位置分析方法**（主策略）：
  - **近252日百分位（最通用）**：<20%=极低波动、20-80%=中性波动、80-95%=高波动、>95%=极值恐慌
  - **OVX自身RSI(14)**：>70=恐慌过度（标准缓解），<30=恐慌不足（标准上升）
  - **方向变化**：单日>5点骤升=恐慌加剧，单日>5点骤降=恐慌缓解
  - **周期选择**：短线用60日百分位，中长线用120/252日百分位
- **绝对阈值分析**（辅助+风控红线）：
  - <20：绝对低波动，市场情绪极度平淡
  - 20-30：历史中性区间
  - >35：一级风控线，波动显著放大
  - >45：二级红线，系统性风险，强制降仓
- **期限结构分析**：近月-远月价差，升贴水结构（Contango/Backwardation）
- **VIX-OVX价差**：OVX相对VIX的溢价程度，衡量原油相对股市的恐慌程度

**指标15：WTI-Brent价差**
- 信号解读：价差扩大/收窄，跨市场套利机会，供需格局变化

**指标16：裂解价差（Crack Spread）**
- 裂解价差类型：
  - 3:2:1裂解价差 = (汽油价格×3 + 柴油价格×2 - 取暖油价格×5) / 5
  - 2:1裂解价差 = (精炼产品价格 - 原油价格) / 原油价格 × 100%
- 信号解读：
  - 高裂解价差 → 炼油利润丰厚 → 炼油厂增加原油采购 → 支撑油价
  - 低裂解价差 → 炼油利润微薄 → 炼油厂减少采购 → 压制油价
  - 季节性：夏季驾车季汽油裂解价差扩大，冬季取暖油裂解价差扩大
- 关键阈值：裂解价差 > 20 美元/桶 = 高利润区，< 10 美元/桶 = 低利润区

**指标17：CFTC持仓报告**
- 持仓类型：
  - 非商业多头（基金做多）：反映投机资金看涨情绪
  - 非商业空头（基金做空）：反映投机资金看跌情绪
  - 非商业净多头 = 非商业多头 - 非商业空头
- 信号解读：
  - 净多头增加 → 投机资金看多 → 支撑油价
  - 净多头减少/净空头增加 → 投机资金看空 → 压制油价
  - 极端净多头（历史高位）→ 警惕反转风险
  - 极端净空头（历史低位）→ 警惕空头回补反弹
- 发布周期：每周六（美国东部时间）
- 报告周期：截至每周二的数据

**指标18：炼油厂开工率**
- 指标含义：
  - 开工率 = 实际加工量 / 设计产能 × 100%
  - 美国平均开工率约 90-95%（季节性波动）
- 信号解读：
  - 高开工率（>93%）→ 炼油厂全力生产 → 原油消耗增加 → 库存下降加速 → 支撑油价
  - 低开工率（<85%）→ 炼油厂减产 → 原油消耗减少 → 库存积累/下降放缓 → 压制油价
  - 开工率上升 → 需求预期改善 → 利好油价
  - 开工率下降 → 需求预期减弱 → 利空油价
- 季节性规律：
  - 春季（3-5月）：检修季，开工率下降
  - 夏季（6-8月）：驾车季，开工率上升至年内高峰
  - 秋季（9-11月）：检修季，开工率下降
  - 冬季（12-2月）：取暖油需求，开工率维持中高水平
- 与库存关系：开工率高 → 库存下降快；开工率低 → 库存下降慢或积累

**指标19：隐含需求（Implied Demand）**
- 计算公式：隐含需求 = 期初库存 - 期末库存 + 净进口 + 国内产量
- 指标含义：
  - 反映市场实际消费能力，不受库存波动干扰
  - 隐含需求 > 实际产量 → 库存下降
  - 隐含需求 < 实际产量 → 库存积累
- 信号解读：
  - 隐含需求上升 → 终端消费强劲 → 支撑油价
  - 隐含需求下降 → 终端消费疲软 → 压制油价
  - 隐含需求与库存变化背离 → 需关注进口/出口变化
- 与库存/开工率联动：
  - 高开工率 + 隐含需求高 → 库存加速下降 → 强烈看多信号
  - 低开工率 + 隐含需求低 → 库存积累 → 强烈看空信号
  - 高开工率 + 隐含需求低 → 炼油产品标准积压 → 开工率将下降

**指标20：库存分结构分析**
- 库存类型：
  - **商业库存（Commercial Inventory）**：炼油厂和管道持有的库存，反映市场供需
    - 商业库存下降 → 炼油需求良好 → 支撑油价
    - 商业库存上升 → 需求疲软/供应过剩 → 压制油价
  - **库欣库存（Cushing Inventory）**：WTI期货交割地的库存，影响WTI期货升贴水
    - 库欣库存高 → WTI期货Contango结构 → 鼓励囤油套利
    - 库欣库存低 → WTI期货Backwardation结构 → 消耗库存
    - 库欣库存接近产能上限 → 预警交割风险
  - **战略石油储备（SPR）**：政府持有的应急储备
    - SPR释放 → 市场供应增加预期 → 短期压制油价
    - SPR补充 → 市场采购增加 → 短期支撑油价
    - SPR处于历史低位 → 政府干预能力受限
- 三者联动分析：
  - 商业库存↑ + 库欣库存↑ → 供需两弱，库存积累
  - 商业库存↓ + 库欣库存↓ → 供需两旺，库存消耗
  - 商业库存↓ + 库欣库存↑ → 库存从库欣转移到商业，期货贴水结构趋向转Contango

**降级策略**：优先访问 https://cn.investing.com/technical/commodities-technical-analysis 获取所有指标；若某指标搜索不到精确数值，使用 WebFetch 抓取 cn.investing.com、TradingView/Investing.com 的技术分析页面，或从已获取的K线数据中计算推导值；优先获取均线、RSI、MACD、ADX、ATR、OVX等核心指标；若核心指标无法获取，标注"技术指标数据暂缺"，跳过综合研判

### Step 2：新闻事件（权威国际源 + 中文辅助）

## 内容要求

### 原油交易员四层分析框架（核心逻辑）

原油是全球政治最敏感的金融资产，分析框架天然是多层次的四层嵌套齿轮：

#### 第一层：供需平衡表（The Foundation）
本质是回答：全球每天多产了多少桶、消耗了多少桶？

| 维度 | 关键变量 | 看什么 |
|------|---------|--------|
| **供给端** | OPEC+产量政策 | 减产/增产配额、履约率、作弊国（超产） |
| | 美国页岩油 | 钻机数（Baker Hughes）、完井数、DUC（未完井）、盈亏平衡成本 |
| | 地缘中断 | 中东冲突→霍尔木兹、利比亚内乱、俄罗斯受制裁后的暗船队流量 |
| **需求端** | 全球GDP增长 | IMF/WTO经济展望；中国/印度/美国三大引擎的炼能变化 |
| | 季节性 | 北半球夏季驾驶季（5-9月汽油需求↑）、冬季取暖油（馏分油） |
| | 替代与效率 | EV渗透率、燃油经济性提升 → 结构性需求压制 |
| **库存** | OECD商业库存 | 最核心的"价格锚"——库存降则涨，累则跌 |

**核心直觉**：油价中长期方向 ≈ 全球液体库存变化趋势。EIA每周发布的库存数据是华尔街最盯紧的数字之一。

#### 第二层：期限结构分析（Forward Curve / Curve Shape）
原油是期货曲线驱动的市场，现货 vs 远期的价差告诉你市场是过剩还是紧缺：

- **Contango**（远月 > 近月）→ 现货过剩 → 储油套利活跃 → 价格承压
- **Backwardation**（近月 > 远月）→ 现货紧缺 → 去库加速 → 价格有支撑

**交易员监控**：
- WTI/布伦特近月价差（spread）
- EFS（Exchange of Futures for Swaps，布伦特-WTI价差）
- 日历价差（Calendar Spreads）——比绝对价格更能反映基本面真实状态

**老手经验**：当曲线从 Deep Contango 翻转为 Backwardation，往往是做多的经典确认信号。

#### 第三层：宏观与美元/利率环境
原油以美元计价，宏观是"引力场"：

| 宏观因子 | 传导机制 |
|---------|---------|
| 美元指数 DXY | 美元走强 → 非美国家购买力下降 → 油价承压（负相关≈-0.5~0.7） |
| 实际利率 | 高利率 → 持有商品机会成本↑ → 商品整体估值压缩 |
| 衰退预期 | 制造业PMI<50持续 → 柴油/航煤需求预警 → 裂解利润缩 → 炼厂降负 → 原油需求塌方 |
| 通胀 | 油价本身是CPI核心项 → Fed政策反噬 → 形成反馈环 |

#### 第四层：资金面与仓位结构（CFTC / COT 报告）
每周五发布的 CFTC Commitments of Traders (COT) 数据揭示：
- 对冲基金/投机资金的 Net Long（净多头） 水位
- 商业套期保值者（Producer Hedging）的空单规模

**极端信号**：当基金经理净多头创多年新高 → 市场"满仓看多"→ 反向脆弱（squeeze risk both ways）；反之空头极值常是底部区域。

**实物流分析（第6轮新增）——平衡表说谎的地方：**

**1. 裂解利润（Crack Margin）= 原油需求的领先信号**
- 原油需求不是消费者直接消耗的，是炼厂买进去、榨出汽油柴油航煤再卖
- 关键指标：3-2-1 crack（3桶原油→2桶汽油+1桶馏分油）、gasoil/diesel crack、jet fuel crack
- **核心逻辑**：裂解利润持续为负比库存数字更早预警油价顶部
- 传导链：原油价格↑↑ → 进料成本太高 + 成品油需求疲软/库存堆高 → 炼厂margin为负 → 降负荷/检修提前 → 原油提货量骤降 → 油价下跌

**2. 成品油库存结构分化 > 原油库存总量**
- EIA周报里别只看第一行"原油库存 ±X万桶"，一定要拆开看：
  - 汽油库存：夏季驾驶季的终端需求温度计；堆太高 = 炼厂要降负
  - 馏分油/柴油库存：工业活动 + 卡车 + heating oil；经济强弱的硬指标
  - 航煤：航空复苏/衰退的前沿
  - 库欣专属库存：WTI交割地，罐容利用率→WTI-Brent脱钩触发器
- **经典陷阱**：原油库存降了但汽油和馏分油双累——看起来"紧"，其实是炼厂还在跑但终端卖不动，下一脚就是炼厂砍负荷、原油反向下跌

**3. 原油品质分化（Quality Differentials）**
- 轻质甜 vs 重质酸：美国页岩是超轻超甜，但全球还有很多炼厂设计吃重酸（委内瑞拉、加拿大、中东重油）
- 当重酸供应受限（制裁/减产）→ 重油折价扩大 → 某些炼厂亏、某些赚
- 关注：Mars（美湾重酸）vs WTI差价、Dubai vs Oman、Urals折扣

## 航运·运费·制裁桶——贸易重构的隐形手（必须分析）

### 油轮运费与保险溢价
原油是全球最依赖海运的大宗商品，运费本质上是从产地到炼厂之间的"隐形税"：

| 指标 | 看什么 |
|------|--------|
| VLCC运费（TD3C：中东→中国） | 中东出货旺不旺、船多还是船少 |
| Suezmax/Aframax区域运价 | 俄罗斯暗流、北海、美洲短途 |
| 战争风险保险溢价 | 红海/霍尔木兹通行成本——这部分不直接减供应，但会让有效供应"变贵" |
| 绕行成本（好望角 vs 苏伊士） | 吨·海里拉长 → 同一桶油占用更多船 → 运费↑ → 到岸成本↑ |

**当前环境特别注意**：制裁执法忽紧忽松，影子船队（dark fleet）承载了相当比例的俄/伊/委出口，一旦OFAC或英国海事执法大面积扣船/禁入港，那些"账面存在的桶"瞬间变成不可达桶。

### 浮式储油（Floating Storage）= 海上隐形库存
岸上库存EIA每周报，但海上浮仓不透明。Kpler/Vortexa卫星追踪显示浮仓可以悄无声息堆到1亿桶+级别——这东西一旦开始卸（价差翻转 or 制裁松动），就是一波隐性抛压。

### 制裁桶 / "条件性供应"——当前时代最特殊的变量
今天的问题不只是"产了多少"，更是"哪些桶被政治/法律卡住了，卡得有多死"：

| 来源 | 逻辑 |
|------|------|
| 俄罗斯 | 生产没崩，但靠暗船队+折扣走亚洲；制裁执法松紧直接决定那300-350万桶/日出口的稳定性 |
| 伊朗 | 出口100-150万桶/日走灰色渠道，一旦美伊谈妥或执法收紧，要么洪水要么断流——两头都是冲击 |
| 委内瑞拉 | 政局突变 = 几十万桶/日的条件性供给开关 |

**实操要点**：不要只盯OPEC+会议，要同时盯——
- OFAC制裁名单更新频率
- G7价格上限执行报告
- 影子船队扣船新闻
- 印度/中国独立炼厂进口窗口（折扣够不够覆盖合规风险）

这些东西的变量名叫policy，不是barrels——它变化速度是外交速度，不是钻井速度，所以市场永远定价不足。

## 中国——全球最大边际买家（最不透明的那个）

### 中国SPR（战略石油储备）补库节奏
中国买不买SPR、买多少、完全不透明，但它能在一个季度内吞吐100-200万桶/日的量级——足够把市场从过剩掰成紧缺。

**跟China相关的雷达**：
- 青岛/舟山等港口卫星浮顶图像（Kayrros/Kpler有）
- 独立炼厂（茶壶）原油进口配额发放节奏
- SC-Brent套利窗口：SC相对Brent异常升水 → 往往暗示东亚现货偏紧/中国在扫货
- 中国成品油出口配额→ 炼厂开工意愿的反向指标

**一句话**：中国因素在原油里约等于"房间里的大象"——你看不到它的账本，但你能从运费、浮仓流向、SC升贴水中看到它的脚印。

## WTI特有的微观结构陷阱——纯技术但能炸账户

### 库欣罐容阈值效应
WTI是唯一主流原油期货做实物交割的（交割地：俄克拉荷马州库欣），所以：

**库欣库存 → 逼近罐容下限**（约2000-2500万桶自由罐容警戒区）
- 空头害怕没地方交货 → 近月被逼空 → WTI与Brent瞬间脱钩（WTI异常高）

**库欣堆满**
- 储油溢价飙升 → 曲线深度Contango → 储油套利锁死库存 → 价格螺旋下行（2020负油价就是这个机制的极端版）

**关键阈值**：每次库欣库存 < 5年均值10%以下或利用率 > 85%，WTI就开始有自己的脾气，不再乖乖跟随Brent。

### 到期移仓（Roll）的机械性扰动
每月临近WTI最后交易日，大量资金被动买卖——尤其是USO类ETF和跟踪基金，它们必须在到期前卖近月买远月：

| 环境 | 移仓效应 |
|------|---------|
| Contango深 | 被动"低卖高买"→ ETF净值额外磨损 → 加剧近月抛压 |
| Backwardation深 | 被动"高卖低买"→ 反而有正carry → 吸引更多inflow → 推近月更高 |

**老手经验**：会在到期前3-5个交易日特别注意WTI近月出现异常波动——那经常是flow，不是fundamental。

## 沙特OSP——OPEC+最即时的"心跳"

每月约25号前后，沙特阿美发布下月对各地区（亚洲/欧洲/美国）的OSP调整：
- **OSP上调** → 沙特觉得市场吃得下、或者故意挺价
- **OSP下调**（尤其对亚洲打折）→ 争夺市场份额的信号

这个比OPEC公报早一周到两周透露真实意图，实货交易员把它当OPEC+的"体温计"。

## 成熟交易员决策检查清单

实际操盘时高手往往有一张 mental checklist：

- [ ] 库存趋势 vs 五年均值 — 是去库还是累库通道？
- [ ] 曲线形态 — Contango还是Backwardation？在加深还是收窄？
- [ ] 炼厂利润(crack) — 裂差是否支撑原油采购意愿？
- [ ] 宏观背景 — 美元方向、衰退概率、Fed姿态
- [ ] 仓位 — 投机资金是否已经极端拥挤？
- [ ] 地缘政治 — 风险溢价的"影子价格"是否已被充分定价？
- [ ] 技术位 — 关键支撑/阻力在哪？（52周高低点、前期成交密集区）

## COT数据源全分类（CFTC交易者持仓报告）

COT（Commitment of Traders）原始数据**唯一官方源头是美国CFTC**，每周五15:30 ET发布，统计当周二持仓，存在3天滞后。

### 一、官方免费数据源

| 数据源 | 地址 | 特点 |
|--------|------|------|
| **CFTC官网报表** | https://www.cftc.gov/MarketReports/CommitmentsofTraders/index.htm | 4类报告（Legacy/Disaggregated/CIT/TFF），Excel/TXT下载 |
| **CFTC Socrata API** | `https://publicreporting.cftc.gov/resource/6dca-aqww.json` | 无密钥免费调用，支持合约代码+日期筛选，JSON输出 |
| **历史批量压缩包** | `https://www.cftc.gov/files/dea/history/fut_disagg_txt_YYYY.zip` | 按年打包，一次性下载多年全市场数据 |

### 二、国内免费可视化（散户首选，无需翻墙）

| 平台 | 入口 | 优势 | 局限 |
|------|------|------|------|
| **金十数据** | 金十官网→机构持仓→CFTC持仓报告 | 全中文、自动算净持仓、折线图、原油/贵金属/外汇全覆盖 | 仅Legacy报告，无杠杆基金细分 |
| **文华财经/博易大师** | 外盘板块内置COT图表 | 与期货盘面联动 | 历史周期短，缺少极端分位指标 |
| **同花顺PC端** | 期货数据栏目 | 中文界面 | 仅基础传统报告 |

### 三、海外免费平台（功能更强）

| 平台 | 特点 | 适用场景 |
|------|------|----------|
| **TradingView** | 社区COT指标可叠加K线，支持Legacy/Disagg/TFF切换 | 技术交易者、量化复盘 |
| **Barchart.com** | 自动计算净持仓占未平仓比例、52周极值区间 | 快速查看极端持仓 |
| **COT-Reports.com** | 纯COT专用站，一键导出CSV | 数据下载 |

### 四、付费专业数据源

| 类型 | 产品 | 特点 |
|------|------|------|
| **情绪分析平台** | Sentimentrader、CotBase、Cotdata.net | COT极值预警、多周期分位、跨品种资金流向 |
| **金融终端** | Bloomberg（`CTOC`函数）、Refinitiv Eikon | 完整细分+衍生品，嵌入全资产分析 |
| **数据服务商** | Quandl（Nasdaq Data Link）、Barchart OnDemand API | 标准时序数据，Python一键读取 |

### 五、程序化API接口

| 接口 | 成本 | 特点 |
|------|------|------|
| **CFTC Socrata API** | 免费 | 无token，限流宽松，适合个人研究 |
| **Cotdata.net API** | 免费层500次/日 | 880+合约，返回净持仓、周度变动、COT分位 |
| **Barchart OnDemand** | 商业付费 | 高并发、低延迟，适合实盘策略 |

### 六、报告类型说明

| 报告类型 | 分类方式 | 适用场景 |
|----------|---------|----------|
| **Legacy** | 商业套保/大型投机/散户（3类） | 快速情绪判断，金十等国内平台默认 |
| **Disaggregated** | 生产商/互换交易商/管理基金/其他（5类） | 商品期货深度分析（原油用此报告） |
| **TFF** | 经纪商/杠杆基金/资产管理/其他（5类） | 金融期货（股指/国债/外汇） |

### 关键提醒

1. **数据滞后**：所有COT统一滞后3个交易日（周二统计，周五发布），短线参考性有限
2. **原油COT**重点关注Disaggregated报告中的Managed Money净多头和Producer净空头
3. **量化回测**优先CFTC官方API（数据最全、格式标准化）
4. **合规**：国内平台仅转发CFTC公开数据，无原始一手增量

## 常见失误速查表

| 失误 | 后果 | 修正 |
|------|------|------|
| 硬编码日期 | 数据过时 | 用相对时间表达（today/yesterday） |
| 断言式喊单 | 违规误导 | 用"若…则…"条件句式 |
| 省略止损价位 | 风险失控 | 每个条件配止损/无效化价位 |
| 重复展示RSI/CCI值 | 冗余 | 操作建议只展示综合信号 |
| 忽略短线/中线分层 | 策略混淆 | 严格区分1-3日和1-4周 |
| 忽略新闻优先级 | 重要信息淹没 | 地缘/OPEC+优先呈现 |
| 时效性不严格 | 结论误导 | 每个数据源确认48h内 |
| 忽略EIA/价差/期限结构 | 分析不完整 | 必须包含核心数据章节 |
