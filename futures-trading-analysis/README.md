# 期货交易分析技能 (Futures Trading Analysis)

## 概述

这是一个商品期货多角色辩论式交易智能体技能，通过调度12个专业角色Agent，执行系统性商品期货分析，输出多维度综合分析报告及操作方案。

## 核心特性

### 两种分析模式
- **单品模式**：分析单个期货品种（如螺纹钢RB、豆粕M、原油SC）
- **产业链模式**：分析一组相关品种（如黑色系：RB/I/J/JM，能源链：SC/FU/LU/BU/PG）

### 5阶段工作流
1. **Phase 1 - 数据收集**（4人并行）
   - 期货技术分析师：价格走势、技术指标、持仓量、期限结构
   - 期货基本面分析师：供需平衡表、库存数据、成本利润
   - 期货新闻分析师：行业政策、天气/地缘事件、产业链动态
   - 期货情绪分析师：CFTC持仓、仓单变化、虚实比

2. **Phase 2 - 投资辩论**（顺序执行）
   - 期货多头研究员：构建买入论证（供应收紧、需求复苏、库存利多）
   - 期货空头研究员：构建卖出论证（供应宽松、需求走弱、库存利空）
   - 期货研究主管：基于供需平衡表、库存周期、期限结构裁决

3. **Phase 3 - 交易决策**
   - 期货交易员：合约选择、移仓换月、保证金管理、虚实比检查

4. **Phase 4 - 风险评估**（3人并行 + 裁决）
   - 激进风险分析师：趋势延续论据、错失成本量化
   - 保守风险分析师：期货特有风险（逼仓、政策突变、流动性枯竭）
   - 中性风险分析师：跨期套利、产业链对冲等温和策略
   - 风险主管：最终风险调整后决策

5. **Phase 5 - 最终报告**
   - 整合所有阶段产出，生成结构化商品期货分析报告

## 使用方法

### 触发词
- 期货分析、商品分析、期货交易
- 多空辩论、风险评估、产业链分析
- 期货投资、期货买卖、期货决策
- futures analysis、commodity trading

### 示例请求
```
帮我分析螺纹钢RB的走势
黑色系整体分析
原油SC该不该做多
豆粕M的多空辩论
```

### 执行模式
- **完整模式**（默认）：执行全部5个阶段
- **快速模式**：用户说"快速分析"/"简要分析"时，仅执行Phase 1 → Phase 3 → Phase 5
- **辩论模式**：用户已提供数据，跳过Phase 1，仅执行Phase 2-5

## 数据源

所有金融数据通过 `neodata-financial-search` skill 获取：
- 禁止使用Yahoo Finance、Alpha Vantage、Tushare、Bloomberg等其他数据源
- 所有Agent均使用此数据源

## 输出格式

### 最终报告包含
1. **最终建议**：BUY/SELL/HOLD决策、信心水平、风险等级、仓位建议
2. **供需平衡判断**：供应端、需求端、库存周期、期限结构
3. **产业链判断**（产业链模式）：整体趋势、品种位置、共振信号、利润分配
4. **四维分析摘要**：技术面、基本面、新闻面、情绪面
5. **投资辩论结论**：多头/空头核心论点、研究主管裁决
6. **风险评估结论**：三方风险观点、风险主管裁决
7. **操作建议**：入场价、目标价、止损价、仓位、合约选择、移仓策略
8. **关键风险提示**：市场风险、品种风险、宏观风险、期货特有风险

## 注意事项

1. **数据源唯一**：必须使用neodata-financial-search skill
2. **决策果断**：研究主管和风险主管必须明确给出Buy/Sell/Hold
3. **并行执行**：Phase 1和Phase 4 Step 4.1使用TeamCreate并行
4. **报告传递**：每个阶段结束后，将完整产出传递给下一阶段
5. **期货特有检查**：持仓量、期限结构、基差、虚实比等
